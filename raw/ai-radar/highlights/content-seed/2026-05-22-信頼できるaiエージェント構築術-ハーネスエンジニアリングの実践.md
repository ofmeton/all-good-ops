# 信頼できるAIエージェント構築術：ハーネスエンジニアリングの実践

## 要約
AIエージェントは長鎖タスクで出力が不安定になり、トークンコストも高騰する課題がある。
この問題に対し、モデル以外の要素を制御する「ハーネスエンジニアリング」を提唱。
ガイド（入力）とセンサー（出力検証）でAIを外部から制御し、信頼性を向上させる。

## Claude 活用 Tip 核
AIエージェントの信頼性を高めるには、モデル自体の改善ではなく、入力ガイド（プロンプト設計）と出力検証（センサー機構）でAIを外部から制御する「ハーネスエンジニアリング」を採用する。
- 適用領域: コード生成, 自動化, データ分析, コンテンツ制作, その他
- 言及ツール: LangChain, API
- スコア: relevance 32 / novelty 18 / applicability 16 = **70**

### 試行プロンプト案
```
以下のタスクを実行するAIエージェントを設計してください。【ガイド】入力時に実行ステップを明示し、制約条件を列挙する。【センサー】出力ごとに検証ルール（形式チェック、値域確認）を設定し、失敗時は再試行ロジックを組む。このハーネス構造でタスク成功率を測定してください。
```
