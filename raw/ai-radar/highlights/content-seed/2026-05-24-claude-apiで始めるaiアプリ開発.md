# Claude APIで始めるAIアプリ開発

## 要約
Claude APIを利用したAIアプリケーション開発の総合ガイドです。
モデルへの直接アクセス「Messages」と、エージェント管理「Managed Agents」の2つの開発アプローチを提供します。
各種プログラミング言語のSDKやAWS Bedrock等との連携もサポートし、開発を支援します。
