# マネーフォワード ME エクスポート（raw）

MF ME web版から chrome-devtools 経由で取得した個人家計データ。raw（immutable）。
取得日: 2026-06-06 / 取得元: ログイン済み個人Chromeセッションに fetch()。

## ファイル

### `cashflow-2020-01_2026-06.csv` — 入出金明細（全期間）
- 取得元: `https://moneyforward.com/cf/csv?from=YYYY/MM/01&month=M&year=YYYY`（月次・ID で重複排除して結合）
- 範囲: 2020-01〜2026-06（71ヶ月にデータ、空月あり）/ 3,743 行
- 元データは **Shift-JIS**（Content-Type は utf-8 と詐称）。本ファイルは **UTF-8 に再エンコード済**
- カラム（10）: `計算対象, 日付, 内容, 金額（円）, 保有金融機関, 大項目, 中項目, メモ, 振替, ID`
  - `計算対象` 1/0（0=集計除外）, `金額` 負=支出, `振替` 1=口座間移動（収支から除外すべき）, `ID` 取引一意キー（dedup 用）

### `asset-history-2025-03_2026-06.csv` — 資産推移
- 取得元: `https://moneyforward.com/bs/history/csv`（期間パラメータなし＝表示中の範囲）
- 範囲: 2025-03-31〜2026-06-06 / 51 行（直近は日次、過去は月末）
- カラム: `日付, 合計（円）, 預金・現金・暗号資産（円）, ポイント（円）`

### 負債（スナップショットのみ・CSV履歴なし）
- 取得元: `/bs/liability` 画面（ダウンロードボタン無し）
- 2026-06-06 時点 負債総額 204,985円（三井住友カード残高 200,189 / ポケットカード 4,796 / 仙台銀行カードローン 0）。住宅ローン等なし＝全額クレカ利用残高
- **履歴は遡れない**。借金トラッキングは今後 `/bs/liability` を定期スナップショットして蓄積する方式が必要。

## 既知の注意点（分析時に必須）
- **2026-04 以降データ激減**: ポケットカード以外の連携が「設定エラー」で停止中。再連携で復活。
- **資産残高が 2026-05〜06 で固定**: 最後の成功取得値で凍結（同上）。
- `大項目=現金・カード`（約 -340万）は ATM 引出等の内部移動を多く含み、`振替` フラグが立っていない行があるため**支出として過大計上されうる**。収支分析時は要個別判定。
- 資産推移の CSV はクライアント表示範囲依存。全期間を取るには事前に「全期間」表示にする必要がある場合あり。
