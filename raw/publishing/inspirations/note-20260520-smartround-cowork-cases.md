# なぜ目に留まったか

- ofmeton のターゲット（非エンジニア・士業・中小事業者）にぴったり
- 業務 × ツール名の組み合わせが具体的: ガイド更新×Cowork / 月次決算×Cowork / チケット整理×Cowork+Linear / デザインレビュー×Cowork+Figma
- Before-After 数値が分かりやすい
  - CS: ガイド更新時間が「半分以下」に短縮
  - 経理: チケット整理 1時間以上 → 20分
  - PdM: 要望チケット処理の自動化実現

# 自分の発信に応用するなら

- 「行政書士 × Claude」「中小工務店 × Claude」みたいに、士業/業種ごとに 3-5 つの活用事例を並べる note 記事の型
- Before-After で時間短縮を必ず数値化
- 4 事例で 8000-12000 字くらいの note 記事に。500円帯の有料記事に良さそう
