# 記帳ガイド（Bookkeeping）

## 概要
個人事業主（青色申告）の仕訳ルール、帳簿フォーマット、月次〆の手順。

## 対象
- bookkeeper（メイン）
- tax-advisor（参考）

---

## 前提
- **申告区分**: 青色申告（65万円控除）
- **収入源**: 業務委託（複数）、Shopify売上、フリーランス報酬
- **経理ソフト**: 未定（当面はスプレッドシート or 手動管理）

---

## 勘定科目（主要なもの）

### 収入
| 科目 | 用途 |
|------|------|
| 売上高 | Shopify売上、フリーランス報酬 |
| 業務委託収入 | RICE CREAM 等の業務委託マネージャー報酬 |
| 雑収入 | アフィリエイト、広告収入、その他 |

### 経費（主要）
| 科目 | 用途 | 例 |
|------|------|-----|
| 通信費 | インターネット、スマホ | 按分あり |
| 消耗品費 | 文房具、小型機器 | |
| 旅費交通費 | 交通費、出張費 | |
| 接待交際費 | 取引先との飲食 | |
| 地代家賃 | 自宅の家賃按分 | 按分率を記録 |
| 水道光熱費 | 電気代等の按分 | 按分率を記録 |
| 支払手数料 | 振込手数料、決済手数料 | |
| 外注費 | 外注への支払い | |
| 雑費 | 上記に該当しないもの | |

### AIサービス費用
| 科目 | 用途 |
|------|------|
| 通信費 or 支払手数料 | Claude API、ChatGPT等のAIサービス |

## 仕訳の基本ルール

### 記録タイミング
- **発生主義**: 取引が発生した日で記録（入金日ではない）
- 請求書発行日 = 売上計上日
- 経費発生日 = 費用計上日

### 仕訳フォーマット
```
日付: YYYY-MM-DD
借方: [科目] [金額]
貸方: [科目] [金額]
摘要: [取引内容の簡潔な説明]
```

### 按分の記録
家事按分がある経費は、按分率とその根拠を明記する。
```
按分率: XX%（根拠: 使用面積比 / 使用時間比）
全額: ¥X,XXX → 経費計上額: ¥X,XXX
```

## 月次〆の手順

### 毎月末に行うこと
1. 当月の全取引を確認（漏れチェック）
2. 未記帳の取引を記帳
3. 入金確認（請求書 vs 入金の突合）
4. 経費の按分処理
5. 月次収支サマリーの作成

### 月次サマリーフォーマット
```
## YYYY年MM月 月次サマリー

### 収入
| 科目 | 金額 | 前月比 |
|------|------|--------|
| 売上高 | ¥XXX,XXX | +X% |
| 業務委託収入 | ¥XXX,XXX | - |
| 雑収入 | ¥X,XXX | - |
| **合計** | **¥XXX,XXX** | |

### 経費
| 科目 | 金額 | 前月比 |
|------|------|--------|
| ... | ... | ... |
| **合計** | **¥XX,XXX** | |

### 差引利益
¥XXX,XXX（目標26万円に対して XX%）
```

---

## 注意事項
- 税務判断に迷う場合は「要確認: 税理士に相談推奨」と明記
- 領収書・請求書の原本保管はユーザーが行う（電子保存法への対応含む）
- 本スキルは記帳の支援であり、正式な税務アドバイスではない
