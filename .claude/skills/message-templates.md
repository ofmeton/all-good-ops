# 文面テンプレート（Message Templates）

## 概要
トーン別の文面テンプレート集。メール・LINE・公式文面の作成時に参照。

## 対象
- message-crafter（メイン）
- freelance-scout（応募文面）
- client-manager（クライアント連絡）

---

## トーンレベル

| レベル | 説明 | 使用場面 |
|--------|------|---------|
| **L1: フォーマル** | 最大限丁寧。初対面・公式・役所 | 官公庁、初取引先、公式文書 |
| **L2: ビジネス** | 丁寧だが簡潔。通常のビジネス | 取引先、クライアント |
| **L3: カジュアルビジネス** | 親しみやすい。関係性ができた相手 | 継続取引先、同業者 |
| **L4: カジュアル** | フレンドリー。知人・友人 | コミュニティ、知人 |

---

## テンプレート

### 1. 初回連絡（L2: ビジネス）
```
件名: [目的]のご連絡 — [自分の名前]

[相手の名前]様

はじめまして。[自分の名前]と申します。
[どこで知ったか/紹介元]を通じてご連絡させていただきました。

[連絡の目的を1-2文で]

[具体的な内容・提案]

ご多用のところ恐れ入りますが、ご検討いただけますと幸いです。
何卒よろしくお願いいたします。

[署名]
```

### 2. お礼（L2: ビジネス）
```
件名: [何に対する]お礼

[相手の名前]様

お世話になっております。[自分の名前]です。

[何に対するお礼かを具体的に]

[相手のおかげで得られた成果や気づき]

今後ともどうぞよろしくお願いいたします。

[署名]
```

### 3. 断り（L2: ビジネス）
```
件名: [件名]について

[相手の名前]様

お世話になっております。[自分の名前]です。

[提案/依頼]について、慎重に検討させていただきました。

大変ありがたいお話ではございますが、[理由]のため、
今回は見送らせていただきたく存じます。

[代替案や将来の可能性があれば]

[相手への配慮の一文]
引き続きどうぞよろしくお願いいたします。

[署名]
```

### 4. 進捗報告（L2: ビジネス）
```
件名: [プロジェクト名] 進捗報告（MM/DD時点）

[相手の名前]様

お世話になっております。[自分の名前]です。
[プロジェクト名]の進捗をご報告いたします。

■ 完了した作業
- ...

■ 進行中の作業
- ...（完了予定: MM/DD）

■ 次のステップ
- ...

■ ご確認・ご判断いただきたい点
- ...

ご不明な点がございましたらお気軽にお知らせください。

[署名]
```

### 5. 日程調整（L3: カジュアルビジネス）
```
件名: 日程調整のお願い

[相手の名前]さん

お疲れ様です。[自分の名前]です。

[目的]について打ち合わせの時間をいただけないでしょうか。

以下の日程で都合の良い時間帯はありますか？
- MM/DD（曜日）
- MM/DD（曜日）
- MM/DD（曜日）

所要時間は[XX分]程度を予定しています。
ご都合の良い日時をお知らせください。

[署名]
```

### 6. LINE返信（L4: カジュアル）
```
[相手の名前]さん、[挨拶]！

[返信内容]

[次のアクションや提案]

[締めの一言]
```

---

## 文面作成時のチェックリスト
- [ ] 相手との関係性に合ったトーンレベルか？
- [ ] 目的が明確に伝わるか？
- [ ] 相手に求めるアクションが明確か？
- [ ] 誤字脱字はないか？
- [ ] 送信先（To/Cc/Bcc）は正しいか？
- [ ] 添付ファイルの漏れはないか？
