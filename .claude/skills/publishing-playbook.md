# 発信プレイブック（Publishing Playbook）

## 概要
SNS・ブログ発信戦略。媒体選定→企画→制作→投稿→分析→改善のサイクルを自律的に回す。月間収益5万円を目標KPIとする。

## 対象
- brand-publisher（メイン）

---

## 収益目標
- **月間収益**: 5万円/月
- **達成時期**: 段階的に（Month 1: 0円 → Month 3: 1万円 → Month 6: 5万円）

## 収益化手段
| 手段 | 説明 | 優先度 |
|------|------|--------|
| アフィリエイト | 商品・サービスの紹介報酬 | 高 |
| 広告収入 | Google AdSense等 | 中 |
| 有料コンテンツ | note有料記事、有料メルマガ等 | 中（実績後） |
| スポンサー | 企業案件 | 低（フォロワー規模後） |

---

## 発信サイクル（6ステップ）

### 1. 媒体選定
**判断基準**:
- ターゲット読者がいるか
- 収益化手段があるか
- 継続運用できるか（工数）
- SEOやアルゴリズムの特性

**推奨アプローチ**: まず1媒体に集中し、軌道に乗ったら横展開。

### 2. コンテンツ企画
**企画テンプレート**:
```
### コンテンツ企画
- テーマ: [タイトル案]
- ターゲット: [誰に向けて]
- 検索ボリューム: [月間検索数]（ツールで調査）
- 競合状況: [競合の強さ・数]
- 差別化ポイント: [自分ならではの切り口]
- 収益導線: [どの収益化手段につなげるか]
- 制作工数: [見積もり時間]
```

### 3. 制作
- writer エージェントに執筆を依頼
- SEO要件（キーワード、見出し構成、内部リンク）を指示に含める
- 画像・図解の必要性を判断
- **冒頭構成は `scqa-writing-framework.md` に従う**（S→C→Q→Aを3段落以内で完結させる）

### 4. 投稿
- **投稿前に必ず人間確認**
- 最適な投稿時間を考慮
- SNSシェアのタイミングを計画

### 5. 分析
**KPI**:
| 指標 | 週次確認 | 月次確認 |
|------|---------|---------|
| PV / インプレッション | ○ | ○ |
| フォロワー数 | ○ | ○ |
| エンゲージメント率 | ○ | ○ |
| 収益 | | ○ |
| 検索順位（主要キーワード） | | ○ |

### 6. 改善
- 分析結果に基づくアクションアイテムを毎月作成
- うまくいったコンテンツのパターンを抽出
- うまくいかなかったコンテンツの原因分析

## コンテンツカレンダー

週単位で管理する。

```
## Week XX（MM/DD - MM/DD）

| 曜日 | 媒体 | コンテンツ | ステータス | 備考 |
|------|------|-----------|-----------|------|
| 月 | ブログ | [記事タイトル] | 企画中/制作中/公開済 | |
| 水 | SNS | [投稿テーマ] | | |
| 金 | SNS | [投稿テーマ] | | |
```

**投稿頻度の目安**（無理のない範囲で）:
- ブログ: 週1-2記事
- SNS: 週3-5投稿

---

## よくある失敗と対策

| 失敗 | 対策 |
|------|------|
| PVだけ追って収益にならない | 企画段階で収益導線を必ず設計 |
| 複数媒体に手を広げすぎ | まず1媒体に集中。3ヶ月は変えない |
| 投稿頻度が維持できない | 無理な計画を立てない。質>量 |
| 個人ブランドの一貫性が崩れる | テーマの軸を3つ以内に絞る |

---

## 注意事項
- SNS投稿・ブログ記事の公開前は**必ず人間確認**
- アフィリエイトリンクの使用は関連法規（景表法、ステマ規制）を遵守
- 他者のコンテンツの無断使用禁止
- 有料コンテンツの価格設定は人間確認必須
