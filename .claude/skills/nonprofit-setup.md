# 法人設立ガイド（Nonprofit Setup）

## 概要
一般社団法人の設立手順をガイドする。2026年内の設立を目標。

## 対象
- nonprofit-advisor（メイン）

---

## 一般社団法人 設立チェックリスト

### Phase 1: 準備（1-2ヶ月）
- [ ] 法人の目的・事業内容を確定
- [ ] 社員（設立時社員）2名以上の確保
- [ ] 法人名の決定（類似名称チェック含む）
- [ ] 主たる事務所の住所を決定
- [ ] 事業年度の決定
- [ ] 設立費用の確保（約11万円〜）

### Phase 2: 定款作成（2-3週間）
- [ ] 定款の記載事項を決定
  - 目的
  - 名称
  - 主たる事務所の所在地
  - 設立時社員の氏名又は名称及び住所
  - 社員の資格の得喪に関する規定
  - 公告方法
  - 事業年度
- [ ] 定款の作成
- [ ] 公証人による定款認証（約5万円）

### Phase 3: 登記（1-2週間）
- [ ] 設立時理事・設立時監事の選任
- [ ] 設立時代表理事の選定（必要な場合）
- [ ] 設立登記申請書の作成
- [ ] 法務局に設立登記申請（登録免許税6万円）
- [ ] 登記完了の確認
- [ ] 登記事項証明書の取得

### Phase 4: 設立後の手続き（2-4週間）
- [ ] 法人口座の開設
- [ ] 法人印の作成
- [ ] 税務署への届出（法人設立届出書）
- [ ] 都道府県・市区町村への届出
- [ ] 社会保険の手続き（従業員がいる場合）

|------|------|------|
| 定款認証 | 約50,000円 | 公証人手数料 |
| 登録免許税 | 60,000円 | 法務局 |
| 印鑑代 | 5,000〜20,000円 | 法人印セット |
| その他 | 10,000〜30,000円 | 謄本取得、郵送費等 |
| **合計** | **約11〜16万円** | 専門家依頼時は別途 |

## 非営利型 vs 普通型

### 非営利型一般社団法人
- 収益事業のみ課税（税制優遇あり）
- 要件:
  - 定款に剰余金の分配を行わない旨を定める
  - 定款に残余財産を国等に帰属させる旨を定める
  - 特定の個人・団体に特別の利益を与えない

### 普通型一般社団法人
- 全所得に課税（株式会社と同様）
- 設立要件はより緩やか

**推奨**: 子どもの居場所づくりという目的を考えると、非営利型を検討。ただし要件を満たせるか確認が必要。

## 注意事項
- **本スキルは手順ガイドであり、法的アドバイスではない**
- 重要な判断（定款の内容、非営利型の選択等）は司法書士・行政書士への相談を推奨
- 法令は変更される可能性があるため、手続き前に最新情報を確認すること
- 全ての提出書類は人間確認必須
