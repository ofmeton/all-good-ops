# all-good-ops

個人の生活・仕事・創作・将来計画を支援する半自律型エージェントチーム。

## 概要

- **30名のエージェント**が8部門 + 横断チームで構成
- **秘書エージェント**が唯一の一次窓口
- **16冊のスキルマニュアル**で品質を標準化
- **評価駆動の自己改善ループ**を内蔵
- **月5,000円以内**のAIコスト制約で運用

## クイックスタート

```bash
cd all-good-ops
# Claude Code でセッションを開始すると、CLAUDE.md が自動的に読み込まれ、
# 秘書エージェントが起動します
```

## ディレクトリ構成

```
.claude/agents/    — エージェント定義（30名）
.claude/skills/    — スキルマニュアル（16冊）
knowledge/         — ナレッジベース
data/              — 運用データ（usage-log, ranks, cashflow等）
scripts/           — 自動化スクリプト
templates/         — テンプレート
outputs/           — 成果物出力先
logs/              — 実行ログ
.github/           — Issue/PRテンプレート
```

## 自動化

| スケジュール | 内容 |
|---|---|
| 毎朝 8:00 | 情報収集 → 文脈更新 → タスク同期 |
| 毎週日曜 | KPI進捗レビュー |
| 毎月1日 | 品質監査・ランク更新 |

## ドキュメント

- `CLAUDE.md` — オーケストレーター（ルーティング・全ルール）
- `CHANGELOG.md` — 変更履歴
