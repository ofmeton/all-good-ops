# 民泊清掃管理アプリ 管理者ガイド

このガイドは、清掃管理業者の運営担当者（管理者）がアプリを使うためのものです。スマートフォン・パソコン両方で使えますが、本ガイドではパソコンでの操作を基本に説明します。

> **アプリのURL:** `<本番URLをここに記入>`
> **本ガイド内のスクリーンショットは納品後に補完予定です**

---

## 1. はじめに

このアプリでできることの全体像です。

- **物件・オーナー・スタッフを登録する** — アプリ上で扱う対象を作ります
- **清掃依頼を作成する** — チェックイン日・チェックアウト日・人数などを入力して依頼を立ち上げます
- **スタッフへ通知を送る** — 依頼が立ち上がると、担当スタッフへ LINE 通知が自動で送られます
- **完了報告を受け取る** — スタッフがチェックリストと写真で完了報告を提出します
- **オーナーへの状況共有** — 確認完了するとオーナーへメール通知され、オーナー専用URLから清掃履歴を閲覧できます
- **備品補充依頼を受ける** — スタッフから備品の不足連絡を受け取ります

操作の流れは「物件・スタッフを登録 → 専用URLを配布 → 依頼を作成 → スタッフが承認 → 開始 → 完了報告 → 内容確認 → オーナーへ通知」が基本の一周です。

## 2. ログイン・ログアウト

### ログイン

1. アプリのURL（上記）をブラウザで開きます
2. 自動的にログイン画面へ案内されます
3. 管理者のメールアドレスとパスワードを入力して「ログイン」を押します

ログイン情報は別途お渡しします。初期管理者のメール・パスワードを変更したい場合は「8. 管理者を追加・管理する」の項を参照してください。

## 3. 物件・オーナー・スタッフを登録する

このアプリでは「物件」「物件オーナー」「清掃スタッフ」をそれぞれ登録し、関係を結びます。

### オーナーを登録する

1. 画面上部のナビゲーションから「オーナー」を選択します
2. 「オーナーを追加」フォームに名前・メールアドレス・LINE ユーザーID（任意）を入力
3. 「オーナーを追加」ボタンを押します

メールアドレスは確認完了時の通知に使われます。LINE ユーザーIDは、オーナーが LINE で通知を受けたい場合のみ入力します（取得方法は「7. 通知について」を参照）。

### 物件を登録する

オーナーを先に登録してから物件を登録します。

1. 画面上部の「物件」を選択
2. 「物件を追加」フォームでオーナーを選択、物件名を入力（住所は任意）
3. 「物件を追加」ボタンを押します

### スタッフを登録する

1. 画面上部の「スタッフ」を選択
2. 「スタッフを追加」フォームに名前・メールアドレス（任意）・LINE ユーザーID（任意）を入力
3. 担当する物件にチェックを入れます（複数可）
4. 「スタッフを追加」ボタンを押します

スタッフは複数の物件を担当できます。物件1軒に対して複数スタッフを担当させることもできます。

## 4. トークンURLを発行・共有する

オーナー・スタッフはこのアプリにログインしません。代わりに、それぞれ専用の「トークンURL」を持ちます。

### スタッフ用トークンURLの発行

1. 「スタッフ」画面で対象スタッフの右側「URLを発行」を押します
2. 発行されたURLが表示されます
3. URL をコピーしてスタッフへ共有してください（LINE で送る・QR コード化して紙で配るなど）

### オーナー用トークンURLの発行

1. 「物件」画面で対象物件の右側「URLを発行」を押します
2. 発行された URL をオーナーへ共有してください

### URLの再発行・無効化

スタッフ・物件それぞれの行に「再発行」「無効化」ボタンがあります。

- **再発行**: 旧URLを無効化して新しいURLを発行します。スタッフが退職した時の引き継ぎ、URLが漏れた時の差し替えに使います
- **無効化**: 単に URL を使えなくします。スタッフが退職する時など

無効化された URL を開くと「このURLは無効です」とエラー表示されます。

## 5. 依頼を作成する

清掃依頼を作成します。

1. 画面上部「依頼」を選択
2. 「依頼を作成」フォームで物件を選択、チェックイン日・チェックアウト日・人数を入力、必要ならオプションメモを記入
3. 「依頼を作成」ボタンを押します

### 入力ルール

- **チェックイン日は翌日以降**でなければ作成できません（当日の急な対応は別途連絡してください）
- **チェックアウト日はチェックイン日より後**でなければなりません
- **人数は1以上**

### 依頼作成後の動き

依頼が作成されると、その物件の担当スタッフ全員に通知が送信されます（LINE 友だち追加済みなら LINE、未登録ならメール）。

担当スタッフが複数いる場合は「早い者勝ち」で誰か1人が「承認」した時点で他のスタッフは承認できなくなります。

## 6. 依頼の流れを把握する

依頼には以下のステータスがあります。

| ステータス | 意味 |
|---|---|
| 未割当 | 依頼を作成した直後。まだ誰も承認していない |
| 割当済み | スタッフが承認した、または管理者が手動で割り当てた |
| 清掃中 | スタッフがアプリで「清掃を開始する」を押した |
| 報告済み | スタッフが完了報告を提出した。管理者の確認待ち |
| 確認済み | 管理者が「内容を確認済みにする」を押した。オーナーへ通知される |
| キャンセル | 管理者がキャンセルした |

通常の流れ:

```
未割当 → 割当済み → 清掃中 → 報告済み → 確認済み
                                        ↓
                               オーナーへ通知
```

「キャンセル」は確認済み・キャンセル以外の状態からいつでも可能です。

## 7. スタッフ割当・キャンセルする

「依頼」画面の各依頼をクリックすると、依頼詳細画面に移動します。

### 手動でスタッフを割り当てる

未割当の依頼や、すでに割り当て済みの依頼でも、管理者は手動で別スタッフを割り当てられます。

1. 依頼詳細画面でスタッフを選択
2. 「スタッフを割り当て」を押します

### 連続予約警告

同じ物件で前後（チェックアウト日 = 次の依頼のチェックイン日、またはその逆）に連続する予約がある場合、依頼詳細画面に黄色い警告が出ます。

> ⚠ 同物件に連続予約があります（...）。割り当て時はスタッフの稼働状況にご注意ください。

これは情報提供のみで、ブロックはしません。同じスタッフが連続で対応するか分けるかなど、状況に応じてご判断ください。

### 依頼をキャンセルする

依頼詳細画面の「この依頼をキャンセル」を押すとキャンセル状態になります。物理削除はせず、履歴として残ります。

## 8. 完了報告を確認する

スタッフが完了報告を提出すると、管理者へメール通知が届きます。

1. 「依頼」画面で「報告済み」ステータスの依頼をクリック
2. 依頼詳細画面の「完了報告」セクションでチェックリスト結果と写真を確認
3. 内容に問題なければ「内容を確認済みにする」ボタンを押します

確認済みにすると、物件オーナーへメール通知が自動送信されます。

### 写真について

- 写真は一覧で表示されます。クリックで拡大可
- 写真の URL は短時間（5分）で失効する仕組みです。アクセスし直すと再発行されます
- 写真は提出から **3ヶ月後に自動削除** されます（容量管理のため）

### 提出内容に問題がある場合

差し戻し機能は本納品の範囲外です。スタッフへ口頭で再対応を依頼してください。

## 9. 備品補充依頼を確認する

スタッフは清掃中に備品の不足をアプリで報告できます。

1. 画面上部「備品」を選択
2. 物件名と日付付きで備品補充依頼の一覧が表示されます

備品補充依頼が出された時、管理者と該当物件のオーナーへメール通知が自動送信されます。

---

## 10. 通知について

このアプリは以下のタイミングで通知を送信します。

| タイミング | 通知先 | 経路 |
|---|---|---|
| 依頼作成 | 担当スタッフ全員 | LINE（友だち未追加ならメール）|
| 完了報告の提出 | 管理者全員 | メール |
| 内容確認済み | 物件オーナー | メール |
| 備品補充依頼 | 管理者・物件オーナー | メール |
| 前日17:00 リマインド | 翌日担当の割当済みスタッフ | メール |
| 24時間未割当アラート | 管理者・物件オーナー | メール |

### LINE 通知が届かない時

- スタッフが LINE 公式アカウントを **友だち追加** していますか? 友だち追加 URL は別途お渡しします
- スタッフ管理画面で **LINE ユーザーID が正しく登録** されていますか?（取得方法は工藤陸へお問い合わせください）
- 上記が問題なくても届かない場合、LINE 通知の月間配信量が無料枠（200通/月）を超えている可能性があります。工藤陸へご相談ください

### メール通知が届かない時

- メールアドレスの登録ミスがないか確認
- 迷惑メールフォルダを確認
- スタッフ・オーナー画面のメールアドレス欄を再確認

## 11. 管理者を追加・管理する

複数の管理者で運用したい場合に使います。

1. 画面上部「管理者」を選択
2. 「管理者を追加」フォームに氏名・メールアドレス・初期パスワード（8文字以上）・権限レベル（通常は 1）を入力
3. 「管理者を追加」ボタンを押します

追加された管理者は、自分のメールとパスワードでログインできます。

### 権限レベル・削除について

本納品の範囲では、権限レベルの変更・管理者の削除は管理画面の UI からは行えません。必要な場合は工藤陸までご連絡ください（API 経由で操作します）。

## 12. 困った時

### よくある質問

**Q: ログインできません**
→ メールアドレス・パスワードを再確認。管理者が複数いれば別の管理者にパスワードリセットを依頼。1人運用で詰まった場合は工藤陸まで。

**Q: 依頼が作成できない（チェックイン日エラー）**
→ チェックイン日は「翌日以降」のみ作成可能です。当日の急な対応は別途お電話でスタッフへご連絡ください。

**Q: スタッフへの通知が届いていないようです**
→ 上記「10. 通知について」の確認項目を順にチェックしてください。

**Q: 写真が表示されない**
→ ブラウザを再読み込みしてください（写真のアクセスURLは5分で失効する仕組みです）。

**Q: 「このURLは無効です」と表示される**
→ URL が無効化されているか、入力ミスです。スタッフ・物件画面で再発行してください。

**Q: 同じ依頼に二重で通知が届いた**
→ 通常起きませんが、起きた場合は工藤陸までご連絡ください（通知ログを確認します）。
