## P12: 2024年 日本広告市場の構造
- 総広告費: 7兆6,730億円（+4.9%）
- インターネット: 3兆6,517億円（+9.6%、全体の47.6%）
- マス4媒体(TV+新聞+雑誌+ラジオ): 2兆3,520億円
- プロモーションメディア: 1兆4,959億円
- ネット広告媒体費: 2兆9,611億円（+10.2%、媒体費ベース）
- 出典: 電通「2024年 日本の広告費」(2025/02/27)

## P13: 運用型広告がマス4媒体を超える構造
- 2021年以降、運用型広告費 > マス4媒体合計
- 2024年: 運用型 約2.5兆円 vs マス4 約2.4兆円
- 運用型比率の継続上昇
- 背景: 動画広告(TikTok/Shorts/Reels)の急伸、自動入札/自動運用の成熟
- ※運用型の定義は調査により差異あり（DSP/SEM/SNS広告を含む）

## P14: リテールメディアの台頭
- Amazon Ads 世界広告売上 2024年: 569億ドル（Google・Meta に次ぐ世界3位）※要確認
- 日本: Amazon Ads / 楽天Ads / Yahoo! ショッピング広告
- 特徴: 購買データに接続された1次データ広告、検索面・商品詳細ページで高CVR
- RMN (Retail Media Network) という業界用語が2024年以降定着
- 副業運用代行としての新たな市場

## P15: 動画広告の急伸

## P16: iOS/ATT シグナル減の系譜
タイムライン図:

- **2021/06**: Meta に対する影響公表（計測困難）
- **2022**: Android プライバシーサンドボックス発表
- **2023/07**: Consent Mode v2 発表

- **2024/07**: Google サードパーティCookie廃止撤回

- 結論: シグナル復元は戻らない、ファーストパーティデータ+サーバーサイド計測が前提

## P17: 🔥Google Cookie廃止"撤回"事件 2024/07/22
- 発表: 2024/07/22 Google Privacy Sandbox 公式ブログ
- 決定: Chrome での3rd Party Cookie廃止を撤回、ユーザー選択UIに切替
- 背景: 初期テストで Google Ad Manager 配信に-34% (publishers) / AdSense -21% の影響
- 規制当局の懸念: UK CMA / EU委員会 / 米DOJ
- **重要**: "撤回" ≠ "計測シグナル減の終わり"。ATT、規制、ブラウザ固有の制限は継続

## P18: 🔥Privacy Sandbox 実質終了 2025/10/17
- 発表: 2025/10/17 Google 公式
- 廃止対象: Protected Audience API (旧FLEDGE), Topics API など大半
- 継続: CHIPS (Cookie 分離), FedCM (認証) のみ
- 理由: 採用率の低さ、規制当局からの継続的懸念
- 結果: 3rd Party Cookie は Chrome に無期限残存
- **広告業界への含意**: Cookieは当面使える、しかし CAPI/SGTM/Enhanced Conversions への移行は継続が業界標準

## P19: Consent Mode v2 (2024/03) 義務化と日本実務
- 2024/03/06 EEA内で Consent Mode v2 義務化
- Google Ads/Analytics で同意信号の送信が必須
- 4つの同意パラメータ: ad_storage / ad_user_data / ad_personalization / analytics_storage
- 日本は義務ではないが、グローバル企業は同実装が事実上標準
- CMP (Cookiebot / OneTrust / usercentrics) 導入案件が2024-2025で急増

## P20: LINEヤフー統合 タイムライン 2023→2026
- **2023/10/01**: LINE/ヤフー等5社合併、LINEヤフー株式会社発足
- **2023/10/02**: コネクトワン構想発表（LINE+Yahoo!のID統合）
- **2024/07**: LINE広告とYahoo!広告のディスプレイ広告プラットフォーム統合開始
- **2024/10/04**: LINEとYahoo!JAPANのアカウント連携開始
- **2025/06/30**: 「Yahoo!JAPANビジネスID」と「LINEビジネスID」統合 →「ビジネスID」
- **2026春**: 「LINEヤフー広告」統合ブランドスタート予定
- LYリーチ: スマホユーザーの94%（重複除く）

## P21: LINEヤフー広告 統合の広告実務インパクト
- 運用管理画面が統合され、LINE/Yahoo!の在庫を横断設定
- ターゲティング: Yahoo!の検索データ + LINEのチャット行動データ
- 広告フォーマット: ディスプレイ(YDA/LAP統合)、検索(Yahoo!)、LINE特有面(トーク上部/LINE NEWS/VOOM)
- 副業運用代行: 日本市場で Google/Meta と並ぶ第3の柱として必須

## P22: 🔥景表法ステマ規制 2023/10/01 施行
- 法的根拠: 景品表示法 第5条第3号 告示（2023/03/28 公布）
- 施行: 2023/10/01
- 対象: 事業者の表示であって一般消費者が広告と判別困難なもの
- 規制対象: 事業者（広告主）。インフルエンサー等の第三者は非対象
- 違反: 措置命令・課徴金
- 実務: PR表記、#PR #広告 #スポンサード #タイアップ などの明示必須
- 副業運用代行: クライアントへの案内必須知識

## P23: EU DSA 2024/02/17 全面適用
- DSA = Digital Services Act
- 施行: 2024/02/17 全EU適用
- 子どもへのターゲティング広告: 全面禁止
- センシティブデータ(人種/政治/性的指向等)でのターゲティング: 禁止
- VLOPs/VLOSEs (超大規模プラットフォーム): 広告レポジトリ公開義務
- X (Twitter) に EUR 45百万円（約70億円）の罰金（2024年）
- 日本からEU向け配信する場合、影響あり

## P24: 米州プライバシー規制の拡大
- CCPA (California, 2020) → CPRA (2023強化)
- 2024-2026: Virginia/Colorado/Connecticut/Utah/Texas/Florida など20州以上で施行
- 米包括プライバシー法 (APRA/ADPPA) は未成立継続
- 実務: CMP (Consent Management Platform) の米州対応必須
- 日本からの越境広告: グローバル CMP + 州別対応が現実解

## P25: 生成AIの広告業界 3波
- **第1波 (2023)**: コピー生成 (ChatGPT/Claude) → 運用者の工数削減
- **第2波 (2024)**: 画像・動画生成 (Midjourney/DALL·E/Sora/Veo) → クリエイティブ制作革命
- **第3波 (2025-)**: 運用AI/エージェント (PMax AI / Advantage+ / Smart+ / Claude Computer Use) → 運用職の役割シフト
- 共通点: 各波で "AIを導く" 人間の介在が減少、意思決定とビジネス翻訳が差別化

## P26: AI検索の勃興 (2024/05-)
- **Google AI Overview**: 2024/05/14 発表、検索結果上部に生成AI要約
- **ChatGPT Search**: 2024/10 リリース、Web検索機能付きチャット
- **Perplexity**: 2023-2024に急成長、直接回答型検索
- **Claude with Web Search**: 2025 リリース
- 影響: 従来の"青いリンク"検索の減少、純粋な情報収集型SEOのトラフィック減少
- ※実測値は媒体により異なる、推計-10%〜-30%

## P27: GEO (Generative Engine Optimization) 胎動
- GEO = Generative Engine Optimization
- 狙い: LLM検索で引用される情報源になる
- 手法:
  - 構造化データ・FAQ形式の強化
  - 権威性の証明（専門家監修、引用元）
  - 明確な因果関係・数字
  - llms.txt の導入（2024末-2025）
- SEO との共通点: コンテンツ品質重視
- 差分: LLMの引用傾向・ソーシャルシグナル (Reddit / Quora) 重要度増

## P28: 代理店業界の再編（2024-2026）
- 電通/博報堂/CARTA/アイレップ/アナグラム: 内製AI化で省人化加速
- Tier2 代理店の統合・淘汰進行
- 運用職 "人月稼働" ビジネスモデルの揺らぎ
- 成果報酬・Retainer + アウトカム連動 の増加
- 副業運用代行: 代理店経由ではなく直接クライアント取引の時代
- ※具体数字は各社 IR 参照、市場観のみ

## P29: 運用職の役割シフトと人材要件
- 旧: 配信設定・入札調整・クリエイティブ指示（手動）
- 新: AI を導く・検証設計・ビジネス翻訳
- 必須スキル 2026:
  - GA4 / SGTM / CAPI の実装理解
  - 生成AIツールのワークフロー化
  - Incrementality / MMM 実験設計
  - ビジネスサイドの LTV / CAC / Payback 会話
- 副業代行: この "新しい運用職" 像で差別化可能
