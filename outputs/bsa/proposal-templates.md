---

## 共通設計

### すべてのテンプレに共通する構造
```
1. 冒頭挨拶 + 案件固有の一文（差別化）
2. 要件の理解を1-2文で示す（誤読リスクを初期に潰す）
3. 解決アプローチ（3点構造）
4. 納期・料金・保証
5. 実績ハイライト（案件ドメインに近いもの）
6. クロージング（質問 or ミーティング提案）
```

### NG 表現リスト
- 「頑張ります」「精一杯対応します」（抽象）
- 「ご縁があれば〜」（弱気）
- 「初心者ですが〜」（信頼失墜）
- 「AI使って早く作れます」（安っぽい。固有価値が伝わらない）

### 推奨 表現
- 「CPA84%削減」などの**数字を含む実績**
- 「72時間以内」などの**具体的な納期コミット**
- 「SLA保証」という**確約の姿勢**
- **クライアント業界の一次理解**を示すキーワード

## T3. LP + 広告運用（広告予算ある層・**継続運用前提**）

> 2026-04-22 改修: market-positioning-analysis.md の戦略C（広告運用主軸）に沿い、継続運用を前面化。制作SLAと運用SLAを分離、成果保証の扱いを明記。

```
〇〇様

ご依頼拝見しました、工藤と申します。
「[案件固有の一文：例：リード獲得LPと広告運用]」について、
**LP制作 + 広告運用を「継続運用前提」で設計**するご提案です。
単発ではなく、2-3ヶ月で成果を可視化する運用を想定しています。

要件整理:
・[商材 / ターゲット / 月間リード目標 / 想定広告費 / 既存計測環境]

アプローチ（3点）:
1. **LPと広告を"同じ頭"で設計**（96時間でLP公開→翌日配信開始）
   → 訴求軸・フォーム設計・CV計測を配信前提で作る
   → CPA84%削減の実績に基づく構造を反映
2. **広告側から順番に打ちきる運用設計**
   → 配信初期は機械学習の学習期間を確保しつつ、広告側の打ち手を1-2ヶ月かけて打ちきる
   → Month 1: 除外KW / 広告文A/Bテスト / 入札戦略（tCPA等）/ マッチタイプ調整
   → Month 2: オーディエンス・時間帯・地域・デバイス入札調整 / 広告表示オプション拡充
   → 広告側の改善余地が見えてきた段階で、必要に応じてLPの改善サイクルに入る
3. **LP改善フェーズでは"爆速・格安"が効く**（制作と運用が同じ人だから）
   → FV・CTA・フォームの部分改修は最短24時間で反映
   → 派生LP制作が必要になれば、既存LPベースの差分制作で**通常の制作会社より格安**（詳細は料金欄）
   → 制作会社への別発注による「1週間待ち・数万円見積もり」が発生しない

料金（二段階）:

【初月セット 100,000円】
- LP制作: 30,000円（制作SLA: 96時間以内公開・超過時20%返金）
- 配信準備: 50,000円（アカウント初期設計・キーワード・広告文3-5本・入稿・入札戦略・CV計測・レポート雛形）
- 初月広告運用: 20,000円（※広告費原価100,000円の場合 / 広告費原価の20%）

【2ヶ月目以降 継続運用】
- **広告費原価の20%/月（月最低20,000円）**
- 日次モニタリング / 週次簡易レポート / 月次レポート＋MTG
- 3ヶ月単位の契約推奨（初月で基盤構築・2-3ヶ月目で効果測定と改善）
- 解約は前月末までの通知で翌月解除

※広告費はクライアント直課金（最低月10万〜、推奨30-50万）。立替は致しません。

SLA（制作と運用で分離）:
- **制作SLA**: LP制作部分のみ。納期超過時は制作料30,000円の20%返金
- **運用SLA**: 稼働保証（レポート期日・返信24h以内・MTG遵守）。
  **広告の成果保証ではありません**（外部要因で変動するため）

実績:
- D2C寝具ブランド Google広告 **CPA84%削減**（3ヶ月運用）
- 広告代理店での月額広告予算数千万規模の運用経験
- BigQuery連携 入稿自動化基盤構築

ヒアリング（30分Zoom）で目標リード数・広告費・商材を深掘りさせてください。
要件定義書＋初月改善仮説を24時間以内にお戻しします。
```

## 運用ガイド

### 1日の提案サイクル
1. **朝8:00-9:00**: 4サイト（Lancers/CW/Coconala/Indeed）の新着案件チェック
2. **9:00-10:00**: fit_score 3以上の案件を5件ピックアップ
3. **10:00-12:00**: 該当テンプレを元に案件固有の一文を入れて5件投下
4. **午後**: 受注案件の制作 or 受注0なら案件追加調査

### テンプレ選択フロー
```
LP単発か？ → T1
コーポレート複数ページか？ → T2
LP + 広告運用が含まれるか？ → T3
既存サイトの修正・改修か？ → T4
建築・不動産・住宅系ドメインか？ → T5（T1/T2と組み合わせ）
```

### 提案後のクロージング率を上げる小ワザ
- **72時間無返信**ならリマインド1通（「ご検討状況いかがでしょうか」）
- **受注→初回MTG設定**は24時間以内に（熱が冷める前に）
- **要件整理**をヒアリング前に箇条書きで先送り（詰まり感の演出）
