# 提案文テンプレート 5パターン（BSA戦略）

> 作成日: 2026-04-22
> 目的: Lancers/Coconala/CrowdWorks に即日投下できる提案文。1件10分以内で調整して送信可能。
> 使用原則: 依頼本文を必ず読み込み、「案件固有の一文」を冒頭に必ず差し込む

---

## 共通設計

### すべてのテンプレに共通する構造
```
1. 冒頭挨拶 + 案件固有の一文（差別化）
2. 要件の理解を1-2文で示す（誤読リスクを初期に潰す）
3. 解決アプローチ（3点構造）
4. 納期・料金・保証
5. 実績ハイライト（案件ドメインに近いもの）
6. クロージング（質問 or ミーティング提案）
```

### NG 表現リスト
- 「頑張ります」「精一杯対応します」（抽象）
- 「ご縁があれば〜」（弱気）
- 「初心者ですが〜」（信頼失墜）
- 「AI使って早く作れます」（安っぽい。固有価値が伝わらない）

### 推奨 表現
- 「CPA84%削減」などの**数字を含む実績**
- 「72時間以内」などの**具体的な納期コミット**
- 「SLA保証」という**確約の姿勢**
- **クライアント業界の一次理解**を示すキーワード

## 運用ガイド

### 1日の提案サイクル
1. **朝8:00-9:00**: 4サイト（Lancers/CW/Coconala/Indeed）の新着案件チェック
2. **9:00-10:00**: fit_score 3以上の案件を5件ピックアップ
3. **10:00-12:00**: 該当テンプレを元に案件固有の一文を入れて5件投下
4. **午後**: 受注案件の制作 or 受注0なら案件追加調査

### テンプレ選択フロー
```
LP単発か？ → T1
コーポレート複数ページか？ → T2
LP + 広告運用が含まれるか？ → T3
既存サイトの修正・改修か？ → T4
建築・不動産・住宅系ドメインか？ → T5（T1/T2と組み合わせ）
```

### 提案後のクロージング率を上げる小ワザ
- **72時間無返信**ならリマインド1通（「ご検討状況いかがでしょうか」）
- **受注→初回MTG設定**は24時間以内に（熱が冷める前に）
- **要件整理**をヒアリング前に箇条書きで先送り（詰まり感の演出）

## 確認ポイント（ユーザーへ）

1. **T1-T5の文体、敬語レベルはこのくらいで良い？** もっと砕ける or 硬めにできる
2. **"CPA84%削減"の実績名義、毎提案で出して良い？** （NDA絡みで心配あれば言い回し変える）
3. **portfolio の "ofmeton.com" URL、正確なドメイン名は？**（仮置きなので要確認）
4. **広告運用の実績(D2Cマットレス3000万等)、どこまで前面に出す？**
5. **「工藤陸」名義と「ofmeton」の使い分け**: 提案は本名、プロフィールはハンドル？
