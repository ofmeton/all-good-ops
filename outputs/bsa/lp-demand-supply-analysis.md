# LP制作市場 需要・供給 両側深堀り分析

> 作成日: 2026-04-22
> 対象: 工藤陸（個人・BSA戦略起点）
> 位置づけ: `market-positioning-analysis.md`（ポジショニング戦略編）の続編。需要サイド（募集案件の業種・規模・サービス内容）と供給サイド（実応募者層の実態）を両面から分析し、今作るべきポートフォリオサンプルLPを絞り込む
> 前提: Month 1-2 広告運用主軸 → Month 3以降 LP本格投入という順序感を崩さない
> SSOT: `wiki/business/bsa/pricing-catalog.md`

## 0. エグゼクティブサマリー

**3行結論**:
1. **需要サイド**: 3プラットフォーム共通で LP制作依頼は「通販・美容・士業/SaaS・スクール・不動産・求人」の6業種に集中。規模は従業員10名以下の個人事業主〜小規模法人が主力。価格帯は5-15万円が主戦場。
2. **供給サイド（実応募者）**: 工藤さんが本当にぶつかるのは認定ランサーTOP層ではなく、**B層（中堅・シルバー下位 30-40%）と C層（駆け出し・スクール卒 25-35%）**。彼らの大半は**テンプレ提案文＋汎用ポートフォリオ**で応募してくる。
3. **実戦略**: A層（TOP層）には時給で勝てない。**B-C層に対して「事業理解＋数値KPI＋広告運用接続」の3点で突き抜ける**ことが勝ち筋。サンプルLPは5本ではなく**3本に絞って深く作る**方がROIが高い。

**サンプルLP再優先度**:
1. 🥇 不動産・リフォーム（portfolio 4本目・既存3本の延長）
2. 🥈 地域BtoB士業（社労士/行政書士等の相談予約LP）
3. 🥉 自費診療クリニック（単科特化・薬機法遵守を参入障壁として活用）

スクール系・BtoB SaaS系は立ち上げ期（〜Month 6）は保留し、広告運用クライアントの業種ミックスを見てから判断。

## 1. 需要サイド分析 ― LP制作依頼の構造

### 1.1 業種別集中度

#### ランサーズ（LP制作 14,991件の業種内訳 / 確度高）

| 業種 | 件数 | 構成比 |
|---|---|---|
| **IT・通信・テクノロジー** | 2,283 | ~15.2% |
| 広告・イベント・プロモーション | 614 | ~4.1% |
| 卸売・小売 | 611 | ~4.1% |
| ビジネス・コンサルティング | 552 | ~3.7% |
| 医療・病院 | 273 | ~1.8% |

→ **IT・SaaS系が圧倒的1位**（2位の3.7倍）。BtoBサービスLPが最大市場。

#### ココナラ（LP制作カテゴリ 約5,000出品者 / 実例ベース）

出品と受注実績が明示的に多い業種：
- **美容・医療系**（美容クリニック、エステ、サロン）
- **女性向け商品・サービス**（コスメ、ダイエット、占い）
- **店舗・飲食系**（整体、飲食店、教室）
- **EC・楽天/Yahoo商品ページ**（BtoC通販）
- **単品リピート通販**（化粧品・健康食品・サプリ）

#### クラウドワークス（案件傾向）
- **EC・SaaS・BtoB企業** が中心
- 業種別統計は非公開、単価帯はランサーズと類似

#### 3プラットフォーム共通：需要が集中する6業種

| ランク | 業種 | 典型LP内容 |
|---|---|---|
| 1 | **化粧品・健康食品・サプリ（D2C単品通販）** | 長尺縦型LP、CVは購入 |
| 2 | **美容・医療（クリニック・エステ・サロン）** | 来院予約、キャンペーン訴求 |
| 3 | **BtoB SaaS・士業・コンサル** | 資料請求・問合せCV、IT業種の中核 |
| 4 | **スクール・セミナー・講座（情報商材含む）** | リード獲得・申込LP |
| 5 | **不動産・建築（モデルルーム・物件・リフォーム）** | 内見予約・資料請求 |
| 6 | **求人LP（介護・飲食・工事系）** | 応募導線 |

### 1.2 発注者の規模

| プラットフォーム | 発注主の主力層 | 根拠 |
|---|---|---|
| **ランサーズ** | 中小企業 + 広告代理店下請 + 個人事業主 | 予算10-20万円帯が中心、企業案件も一定割合 |
| **ココナラ** | **個人事業主 + 小規模法人**が最多 | 個人発注で5万円未満が54.4%、5-10万円が31.2% |
| **クラウドワークス** | 中小企業 + EC運営者 | 平均80,000円/ページ、納期20日 |

**共通パターン**: **従業員10名以下の小規模事業者が圧倒的多数**。大企業案件はほぼ入らない（代理店経由になる）。

### 1.3 依頼内容（サービス範囲）の分類

#### ランサーズ 14,991件の業務範囲内訳

| 範囲 | 件数 |
|---|---|
| デザイン（のみ含む） | 6,216 |
| コーディング | 4,980 |
| 企画・デザイン構成 | 3,048 |
| ライティング | 1,970 |

→ **「デザインのみ」案件が最多**。代理店や内製コーダー持ちの発注が多い。

#### 3プラットフォームで共通する3大依頼パターン

| パターン | 価格帯 | 特徴 |
|---|---|---|
| **A. デザインのみ（PSD/Figma/AI納品）** | 3-15万円 | 代理店下請け、コーディングは別発注 |
| **B. 単一セクション・パーツ（FV・ヘッダー・画像）** | 3-6万円 | 短納期、リピート入口になりやすい |
| **C. 業種特化・フルLP（デザイン+コーディング）** | 8-30万円 | 医療・不動産・BtoB系、**継続依頼が発生しやすい** |

#### ツール指定の傾向（新しい変化）
- **STUDIO / ペライチ / Wix / Figma対応**の指定が急増
- WordPress案件は減少傾向
- ココナラAIカテゴリは前年比394%成長

### 1.4 類似点と相違点

#### 3プラットフォーム共通の6パターン
1. **業種集中**: 通販・美容・士業/SaaS・スクール・不動産・求人で大半
2. **規模**: 個人事業主〜小規模法人が主力、大企業はほぼ不在
3. **価格レンジの中心**: 5-15万円が全プラットフォーム共通の主戦場
4. **1ページ構成のLP**が圧倒的多数（長尺縦型）
5. **スマホ優先（レスポンシブ必須）**: BtoCでは特に顕著
6. **広告運用は別発注が通例**: LP制作案件で運用まで言及されることは少ない

#### 相違点

| 観点 | ランサーズ | ココナラ | クラウドワークス |
|---|---|---|---|
| 価格帯の中心 | 10-20万円 | **3-15万円（最安値層厚い）** | 8-30万円 |
| 発注フロー | プロジェクト型（相見積） | **出品型（固定パッケージ）** | プロジェクト型 |
| 発注者層 | 中小企業・代理店多め | **個人事業主・BtoCが多め** | 中小企業・EC事業者 |
| BtoB比率 | 高（IT系2,283件） | 中（金融71件など限定的） | 中 |
| 人気ツール | WordPress・Figma | STUDIO・ペライチが目立つ | WordPress・コーディング系 |

## 2. 供給サイド分析 ― 実応募者の実態

### 2.1 n数の注意書き

完全網羅には各プラットフォームの全案件の全応募者を調査する必要があり、応募者個別情報は発注者しか見られない仕様。今回の一次データは：

- ランサーズ実案件ページ: 提案数が見えるもの 数十件を確認
- 応募者側のnote/ブログ記事: 数十本
- プラットフォーム公式の提案・ランク仕様
- スクール卒業生・副業ブログの体験談

**以下は「定量×定性の混合推定」で精度は±20%。裏取り前提で扱う。**

### 2.2 提案数の定量スナップショット

| 案件タイプ | 典型提案数レンジ | 根拠 |
|---|---|---|
| 予算10-20万円・標準LP | **15-25件** | 実在案件「スタッフ応募LP 20万円」で22件 |
| 予算30万円超・急募・実績必須明記 | 5-15件 | 実在「急募オプションLP 30-50万」でも応募薄く留まる |
| 予算5-10万円・初心者OK系 | **30-100件** | 初心者が殺到、Web制作系で「30件に1件受注」 |
| コンペ型LP | 10-50件 | コンペは応募がさらに集中 |

**新規出た直後の案件は 0提案、24時間以内に10-20件 が標準**

### 2.3 応募→受注確率

| 層 | 提案通過率 |
|---|---|
| 初心者（実績0-10件） | **1-5%**（15-20件提案で1件受注） |
| 中堅（シルバー相当） | 推定 5-15% |
| 認定ランサー | 推定 20-40% |
| Web制作全般 | 約3.3%（30件応募で1件） |

**実在案件の例**: 予算20万円・提案22件 → 採用1件。採用者は**認定ランサー・実績381件・5年連続Lancers of the Year ノミネート・時給3,750円〜**。選定基準は「過去LPポートフォリオ提示」

### 2.4 実応募者の層別構成（推定）

1案件に15-25件提案されるときの内訳（n=直接観測10案件＋応募者側ブログ20本の混合推定）：

| 層 | 構成比 | プロフィール像 | 提案の質感 | 価格戦略 |
|---|---|---|---|---|
| **A層**: 認定ランサー・シルバー上位 | 15-25% | 実績100件超・評価4.9+・業種1-2特化 | 案件要件に沿った個別提案・ポートフォリオURL添付 | 相場通り〜やや高め |
| **B層**: 中堅・シルバー下位 | **30-40%** | 実績20-80件・評価平均・汎用デザイナー | 半テンプレ＋案件名差し替え程度・提案文中位 | 相場の下限 |
| **C層**: 駆け出し・ブロンズ/レギュラー | **25-35%** | 実績0-15件・Web制作スクール卒・ポートフォリオ架空案件3本 | テンプレ丸コピペ・案件名すら差し替え無しも混在 | 相場の50-70% |
| **D層**: ダンピング系 | 5-15% | 実績0-5件・「実績作りのため相場の半額」宣言 | 安さだけ訴求 | 相場の30-50% |

#### 観測されたパターン
- 件数の過半は **B層＋C層**
- 発注者は「使い回し提案は見抜く」が前提。それでもテンプレ提案が多いのは、A層以外の大半がテンプレに頼っているから
- 採用者はほぼ常に A層。**競合で"落ちる"のは B層同士、C層同士の戦い**

### 2.5 実応募文の中身の画一性

副業ブログ・提案文テンプレ販売記事を集約した結果：

| 構成要素 | 含有率 |
|---|---|
| 冒頭の挨拶 | ほぼ全員 |
| 自己紹介（経歴） | 8-9割 |
| 実績件数の列挙 | 7-8割 |
| ポートフォリオURL | 6-7割 |
| 案件内容への言及 | 半数は一般論止まり |
| **案件固有の提案・アイディア** | **2-3割未満** |
| **数値KPI・成果への言及** | **1割未満** |
| **発注者の事業理解** | **1割未満** |
| **広告運用まで含む提案** | **1割未満** |
| 納期の具体化 | 半数 |
| 価格の具体化 | 半数 |

**示唆**: 「案件固有の事業理解＋数値KPIの見立て＋広告運用まで接続」の3点を全提案に入れるだけで、**提案者の上位10%に入れる**。

## 3. 工藤さんが本当にぶつかる相手 ― 戦略の含意

### 3.1 真の競合は B層と C層

A層（認定ランサー）は同じ案件に応募していても、発注者視点では実質的にA層内での採用争い。B・C層は提案フェーズで弾かれる構造。

したがって：
- **A層には「実績数」で勝てない**（時給で戦えば消耗）
- **B・C層に対する武器**:

| 武器 | B層に対して | C層に対して |
|---|---|---|
| CPA84%削減の数値実績 | **致命的に差をつけられる** | B層以上に差がつく |
| 元広告代理店の肩書 | 差別化効く | 差別化効く |
| 業種特化サンプルLP | **B層の汎用ポートフォリオを上回る** | 完勝 |
| 広告運用まで受ける | **B層にはほぼ無い訴求** | 完勝 |
| AI活用の透明開示 | 差別化効く | 差別化効く |
| 個別カスタム提案文 | B層のテンプレ提案を上回る | 完勝 |

### 3.2 実戦略の定式化

> **「A層とは住み分け、B-C層は提案の質で突き抜けて勝ち切る」**

A層と同じ案件に落ちるのは構造上避けられないが、A層は1案件に1人しか採用されないので、**採用されなかったB-C層 15-20件に差をつければ、発注者の「A層予算が無い時の次善の選択」になれる**。

### 3.3 既存 `proposal-templates.md` との整合

既存テンプレは方向性として正しいが、今回の観測を踏まえると：
- **冒頭3点のうち1つを「事業理解の要約」に固定**（B-C層ほぼ不在）
- **数値KPIの見立てを毎回1つは入れる**（B-C層ほぼ不在）
- **広告運用への接続文を末尾ではなく本文中盤に**（B層にも少ない）

## 4. ポートフォリオサンプルLP 再優先度

### 4.1 補正の論理

前回は5本提案したが、今回の供給サイド分析を踏まえると：

1. **5本作るより3本を深く作る方が B-C層との差別化が立つ**（B-C層は「業種特化を名乗りつつ中身が汎用」が多いので、逆手を取る）
2. **Month 1-2広告運用主軸、Month 3+LP本格投入**の順序を崩さない → LPサンプルは広告運用の「証拠物件」として機能する形に
3. **既存portfolio 3本（建築系）の延長サンプル4本目は必ず作る**（最速・最低コスト・最大説得力）
4. 広告運用との接続ストーリーが語れない業種のサンプルは優先度が下がる

### 4.2 3本絞り込み

#### 🥇 優先度1: 不動産・リフォーム（portfolio 4本目）

**誰向け**: 地域の工務店・リフォーム会社・注文住宅ビルダー。Googleリスティング＋SNS広告で資料請求/モデルハウス見学予約を獲得している層

**サンプル内容の要点**:
- 見出し: 「◯◯（地名）の注文住宅｜ZEH対応・総額△△万円〜」
- 信頼要素: 完成事例・お客様の声・施工エリアマップ・構造保証
- CV: 来場予約・カタログ請求・オンライン相談
- 差別化: 「広告運用で資料請求CPA◯◯円達成」の実績クレームを仕込む

**選定理由**:
- **既存portfolio 3本（minato/hiyori/numata）の延長で作成コストが最も低い**
- 業種特化の深さで一気に競合差をつけられる（3→4本で「建築専門LP」を名乗れる）
- Month 1-2の地域広告運用→Month 3で資料請求CV改善という定番ペア
- 発注者層（地域中小企業オーナー）に対する工藤さんの既存経験値が最も高い
- B-C層で建築4本持ちは皆無

**作成コスト**: 低（既存デザインシステム流用可）

---

#### 🥈 優先度2: 地域BtoB士業（社労士 or 行政書士）相談予約LP

**誰向け**: 従業員3-10名の地域密着型士業事務所。指名キーワード＋地域キーワードで広告運用中、問い合わせから相談予約への歩留まりを改善したい層

**サンプル内容の要点**:
- 見出し: 「◯◯（地名）の社労士｜助成金申請 相談30分無料」
- 信頼要素: 実績数・有資格者数・対応企業ロゴモザイク・受賞/連載
- CV: LINE予約・Webフォーム・電話の3導線
- 下部: よくある質問・料金目安・スタッフ顔写真

**選定理由**:
- ランサーズ業種分布で「ビジネス・コンサル」552件の主要サブセグメント
- 広告運用との接続性最強（地域×指名は運用型広告の定番）、Month 3 で「次はLP改修しましょう」が自然に出る
- 継続運用化率が高い（月の広告費が読めるため月次伴走契約が成立しやすい）
- 既存portfolio（建築中小企業向け）と発注者像が近似、工藤さんの営業文脈が共有できる
- B-C層の汎用LPデザイナーで士業特化は不在

**作成コスト**: 中（業種調査とテンプレ化が必要）

---

#### 🥉 優先度3: 自費診療クリニック（単科特化）予約LP

**誰向け**: 歯科の自費診療（インプラント・矯正）、眼科のICL、整形外科のヒアルロン酸 等。地域×症状キーワードで広告運用中、CPA高止まりで困っている層

**サンプル内容の要点**:
- 見出し: 「◯◯（症状）でお悩みの方へ｜△△クリニック（エリア名）」
- 信頼要素: 医師経歴・症例数・使用機器・Before/After（薬機法に抵触しない範囲で）
- CV: 初診予約・カウンセリング無料相談・LINE
- ❗ 薬機法・医療広告ガイドライン遵守を明示（これ自体が訴求になる）

**選定理由**:
- 単価が最も高い業種帯（月広告費50-200万円が普通）、L3継続運用で時給換算1万円超が現実的
- 広告代理店出身の工藤さんにとって薬機法・医療広告ガイドライン遵守は得意領域（B-C層が避ける参入障壁）
- 月1-2社継続で月収KGIの半分を作れる破壊力
- 競合のLP特化プレイヤーで「自費診療×単科」まで絞り込む層は空白

**作成コスト**: 中〜高（薬機法調査が必要）

### 4.3 保留にする2案

| 保留案 | 保留理由 | 再検討タイミング |
|---|---|---|
| スクール・セミナー申込LP | 情報商材との隣接リスクあり、選別コスト高。作成コスト低いが立ち上げ期のブランド毀損リスク | Month 4-6 で真面目系スクール案件が実際に来たら着手 |
| BtoB SaaS 資料請求LP | 市場最大（IT 2,283件）だが、工藤さんの既存実績との接続が弱く作成コスト高 | Month 3以降の広告運用クライアントにIT系が混ざってから判断 |

### 4.4 推奨実施スケジュール

| 時期 | 作業 |
|---|---|
| **Wk1（今週）** | 不動産・リフォーム4本目の要件仕様書作成 |
| **Wk2** | 不動産4本目のデザイン・コーディング着手 |
| **Wk3-4** | 不動産4本目完成 → portfolioサイト追加 |

## 5. 今回分析の最大の学び

**「TOPワーカー分析」よりも「実応募者の分布と訴求パターン」の方が、工藤さんの現場と地続き**。

B-C層のテンプレ提案がここまで厚いなら：

- **サンプルLP 3-5本を作るよりも、提案文のカスタム化と業種別ヒアリング設計を先に仕込む方がコスパが高い可能性**がある
- サンプルLPはあくまで「信頼の証拠」。本丸は「提案1本目の瞬間に発注者を掴む個別カスタム提案」
- 3本サンプルを作る間に、各業種のヒアリング質問テンプレ＋提案の「事業理解見せ方パート」を並行整備するのを推奨
