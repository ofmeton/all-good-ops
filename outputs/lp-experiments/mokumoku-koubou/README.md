# みどり工務店 LP（リフォーム / 注文住宅 サンプル）

工務店 / リフォーム業向け LP のサンプル実装。架空の会社「みどり工務店」を題材に、AI 画像生成を活用したオリジナルキャラクター・写真と、温かみのある絵本タッチで構成。

## 特徴

- **AI 画像生成 + 手描き水彩タッチ**: ブランド統一されたキャラクター・施工写真
- **横スクロール無限マーキー**: お客様の声を「数の多さ」で訴求
- **施工事例カテゴリフィルタ**: 注文住宅 / リノベ / リフォーム の絞り込み
- **FAQ アコーディオン / お問い合わせバリデーション / 7 ステップ可視化**
- **スマホファースト**: FV 100svh、SP 固定 CTA、装飾の被りに配慮した可読性設計

## 技術

- React 18 (UMD) + Babel standalone
- CSS Houdini `@property` でアニメーション補間
- Vercel デプロイ対応（静的ホスティング、ビルド不要）

## ローカル動作確認

```sh
python3 -m http.server 8770
# → http://localhost:8770/
```
