# 配送設定 UI セットアップガイド（TPCBCグッズ統合プロフィール）

配送設定は API スコープ制限のため管理画面で操作。
場所: 管理画面 → 設定 → 配送と配達（Settings → Shipping and delivery）

設計方針:
- TPCBCグッズ4種を **1プロフィールに統合**（海外は合算でEMS1回分）
- 重量帯の境界を商品に合わせ、国内の配送方法を正しく振り分け
  - 小物(ステッカー30g/アクキー50g) → クリックポスト相当
  - Tシャツ(250g) → レターパック相当
  - パーカー(700g) → ゆうパック相当（地域別）
- 海外（台湾・韓国・中国）= EMS第1地帯 重量別

---

## Step 1: グッズ統合プロフィール作成
1. 新規プロフィール作成、名前: `TPCBCグッズ（国内＋海外EMS）`
2. 以下4商品を追加（旧プロフィール クリックポスト/レターパック/TPCBCパーカーゆうパック から移動）:
   - TPCBC 波形ステッカー / TPCBC アクリルキーホルダー / TPCBC 波形Tシャツ / TPCBC ジップアップパーカー
3. 空になった旧3プロフィールは削除

---

## Step 2: 日本ゾーン（地域別 × 重量別）
日本国内を下記5ゾーンに分け、各ゾーンに **5つの重量別料金** を設定。
重量境界（450g / 1300g / 2000g）を商品の組合せに合わせ、配送方法を正しく振り分ける。

### 重量帯 → 配送方法（全ゾーン共通の考え方）
| 重量範囲 | 配送方法 | 該当する組合せ例 |
|---|---|---|
| 0〜149g | クリックポスト ¥185（全国一律） | ステッカー/アクキー単品 |
| 150〜449g | レターパックプラス ¥600（全国一律） | Tシャツ1枚 |
| 450〜1299g | ゆうパック60サイズ（地域別） | Tシャツ2枚 / パーカー1枚 / パーカー1+Tシャツ1 |
| 1300〜1999g | ゆうパック80サイズ（地域別） | パーカー2枚 |
| 2000g以上 | ゆうパック100サイズ（地域別） | パーカー3枚〜 |

### 地域ゾーンとゆうパック料金（神奈川発）
各ゾーンに「クリックポスト¥185」「レターパック¥600」＋下記3サイズを入れる。

| ゾーン名 | 含める地域 | 60サイズ(450-1299g) | 80サイズ(1300-1999g) | 100サイズ(2000g-) |
|---|---|---|---|---|
| 本州東＋東海（神奈川含む） | 関東・南東北・北東北・信越・北陸・東海 | ¥880 | ¥1,200 | ¥1,500 |
| 近畿 | 滋賀・京都・大阪・兵庫・奈良・和歌山 | ¥990 | ¥1,310 | ¥1,620 |
| 中国・四国 | 鳥取・島根・岡山・広島・山口・徳島・香川・愛媛・高知 | ¥1,150 | ¥1,440 | ¥1,780 |
| 北海道・九州 | 北海道・福岡・佐賀・長崎・熊本・大分・宮崎・鹿児島 | ¥1,410 | ¥1,710 | ¥2,020 |
| 沖縄 | 沖縄 | ¥1,450 | ¥1,810 | ¥2,160 |

※ クリックポスト¥185・レターパック¥600 は全ゾーン共通。
※ 重量境界は推定重量（Tシャツ250g/パーカー700g）前提。実測登録後に境界を要再確認。

## Step 3: アジアゾーン（海外）= EMS第1地帯 重量別
1. ゾーン追加、名前: `アジア（台湾・韓国・中国）`、国: **台湾 / 韓国 / 中国**
2. 重量別料金（合算カートでも1回分で計算される）:

| 重量範囲 | 料金 |
|---|---|
| 0〜0.5kg | ¥1,450 |
| 0.5〜0.75kg | ¥1,750 |
| 0.75〜1.0kg | ¥2,200 |
| 1.0〜1.25kg | ¥2,500 |
| 1.25〜1.5kg | ¥2,800 |
| 1.5〜2.0kg | ¥3,400 |
| 2.0〜3.0kg | ¥4,400 |
| 3.0〜5.0kg | ¥6,400 |
| 5.0kg以上 | ¥10,600 |

※料金は各帯上限の公式EMS額＝下回らない（赤字回避）。全25区分の正確版は `ems-zone1-rates.md`。

### 重量帯と商品の対応（確認用）
- ステッカー30g/アクキー50g → 0.5kg帯 ¥1,450
- Tシャツ250g → 0.5kg帯 ¥1,450
- パーカー700g → 0.5〜0.75kg帯 ¥1,750
- Tシャツ+パーカー(950g) → 0.75〜1.0kg帯 ¥2,200（合算1回）
