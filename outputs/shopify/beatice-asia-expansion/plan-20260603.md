# BEAT ICE（山口智史ツアーグッズEC）アジア3カ国 越境EC化 実行計画

- 作成日: 2026-06-03
- 対象ストア: beatice.jp（`beatice.myshopify.com` / 株式会社BEAT ICE）
- 対象国: 台湾・韓国・中国（中国は越境発送＋Alipay/カードで割り切り並行）
- 操作手段: Shopify CLI（`shopify store execute` で Admin GraphQL）。決済プロバイダ有効化等は管理画面UI（手動）
- 担当: secretary 主導 → 実作業 business-ops/shopify-operator、現地語コピー communication/message-crafter

---

## 0. ゴール定義

「台湾・韓国・中国のファンが、現地語・現地通貨で迷わず購入でき、返品/配送/法定表記が越境前提で整備された状態にして、販売準備を完了させる」

成功条件:
1. 4言語（日/韓/簡/繁）で商品・ナビ・チェックアウト・ポリシーが破綻なく表示
2. 海外発行カードで決済が通る（＋可能なら Alipay+/PayPal）
3. 返金・配送・利用規約・特商法が越境前提で整備・4言語掲載
4. 台湾・韓国・中国への配送ゾーン／送料が設定済
5. 各国テスト注文がチェックアウトまで成功

## 2. 確定した方針（意思決定済）

- アクセス: Shopify CLI 経由（認証済）
- 市場: 分割せず1市場のまま中身整備
- 翻訳: ファン向け丁寧仕上げ。「棚田」等コンセプト語は日本語＋現地語注釈で残す。1リソースずつ擦り合わせ
- 決済: **海外発行カードを受付るに変更**（要・株式会社BEAT ICE 承認）。Alipay+/PayPal は追加検討
- 中国: 越境発送＋カード/Alipayで割り切り（ICP等は追わない）

## 3. 実行フェーズ

### Phase 1: 決済・法定表記の是正（最優先・ブロッカー解除）
1. 決済設定で海外発行カード受付を有効化（管理画面UI・要会社承認）
2. Alipay+ / PayPal の追加可否を確認・検討（手動）
3. 特商法テキスト改訂（`shopPolicyUpdate` LEGAL_NOTICE）
   - 「海外カード不可」削除、越境発送・関税負担・返品条件を越境前提に書換
   - 連絡先住所の正規化
4. 完了基準: 海外カードでテスト決済が通る＋特商法が越境前提

### Phase 2: ポリシー整備（4言語）
1. 返金ポリシー新規作成（`shopPolicyUpdate` REFUND_POLICY）: 不良品対応／客都合返品の可否／返品送料・関税負担(DDU)を明記
2. 配送ポリシー新規作成（SHIPPING_POLICY）: 国際配送地域・送料・配達日数・関税
3. 利用規約（TERMS_OF_SERVICE）作成
4. プライバシー・連絡先の越境対応微修正
5. 上記すべてを ko / zh-CN / zh-TW に翻訳登録（`translationsRegister`）
6. 完了基準: フッター全ポリシーが4言語で実体表示（404解消）

### Phase 3: 配送設定
1. deliveryProfiles で台湾・韓国・中国の配送ゾーン・送料を設定（管理画面/API）
2. 関税方針（DDU=着払い前提）を配送ポリシーと整合
3. 完了基準: 3カ国宛でチェックアウトに送料が出る

### Phase 4: 翻訳仕上げ
1. translatableResources で未翻訳・MT不自然箇所を洗い出し
2. テーマ英語残存（"OUR BEST SELLER"/"Show filters"等）を翻訳
3. 商品名・コピー・固有名詞をユーザーと擦り合わせ（「棚田」等は日本語＋注釈）
4. `translationsRegister` で反映
5. 完了基準: 4言語で破綻・英語残存なし、固有名詞のニュアンス確定

### Phase 5: 販売準備チェック
1. 各国（geo）でのテスト注文（言語/通貨/送料/税/決済/通知メール多言語）
2. hreflang・SEO確認
3. 最終公開判断（会社確認）

## 5. 技術メモ（Admin GraphQL）

- ポリシー更新: `shopPolicyUpdate(shopPolicy: {type, body})`
- 翻訳登録: `translationsRegister(resourceId, translations: [{locale, key, value, translatableContentDigest}])`
- 翻訳対象取得: `translatableResource(resourceId)` / `translatableResources(first, resourceType)`
- 市場: `markets`, `marketUpdate`, `marketWebPresenceUpdate`
- 配送: `deliveryProfiles`, `deliveryProfileUpdate`
- 実行: `shopify store execute --store beatice.myshopify.com --query '...' [--allow-mutations]`
