## 2. テーマ①：道管と師管（17:05-17:20）

### 一緒に描く図のプラン

白紙の **左半分** を使う。

1. まず**茎の縦切り**を描く（縦長の長方形）。中に縦に 2 本の管を描く。**位置はあえて自分で先に描かず、つかさに「2本のうちどっちが内側か覚えてる？」と聞く** → 答えが出なくてもOK、「内側が道管」と図に書きながら入れる
2. 次に**横断面（輪切り）の円**を右に並べて描く。維管束を 5-6 個リング状に配置（双子葉のパターン）。**内側=道管・外側=師管** をマーカーで色分け（青=道管、赤=師管 推奨）
3. 矢印で**流れるもの**を入れる：
   - 道管（青）: **下から上**へ。「水＋無機養分（根が吸ったもの）」
   - 師管（赤）: **上にも下にも**（双方向）。「光合成でできた糖などの養分」
4. 図の下に**3点メモ**を書かせる：
   - 道管 = 内側 = 水と肥料（無機養分） = 一方通行（根→葉）
   - 師管 = 外側 = 糖（光合成産物） = 双方向
   - 道管＋師管 = **維管束**

### 言葉の入れ方（つかさへの問いかけ順）

1. 「植物って、水を根から吸って葉まで届けてるよね。これって誰が運んでると思う？」
2. （道管と答えられたら） 「じゃあ葉っぱで作った栄養はどうやって運ぶ？」
3. （師管と答えられたら） 「じゃあ図描いてみよう。茎の中ってどうなってると思う？」

→ 答えに詰まったら**すぐ図に戻って一緒に描く**。詰まり時間を 30 秒以上作らない。

### 暗記タグ（最後にチラッと渡す）

- 「**道**は **水**道の道、水が流れる」
- 「**師**は **師**匠の師、師匠は栄養くれる人」← イメージ語呂

## 3. テーマ②：呼吸と光合成（17:20-17:35）

### 一緒に描く対比表のプラン

白紙の **中央** に縦線を引いて、左=光合成、右=呼吸の T 字表を作る。

| 項目 | 光合成 | 呼吸 |
|---|---|---|
| 時間帯 | **明るい時だけ** | **24時間ずっと** |
| 吸う気体 | CO₂（二酸化炭素） | O₂（酸素） |
| 出す気体 | O₂（酸素） | CO₂（二酸化炭素） |
| エネルギー | **作る**（光エネ → 化学エネ） | **使う**（化学エネ → 活動） |
| 場所 | 葉緑体（緑のとこ） | ミトコンドリア（全細胞） |
| 必要なもの | 光・CO₂・水 | 有機物（糖）・O₂ |

**描く順**: 項目名は先にこちらが書く。**中身の各セルをつかさに埋めてもらう**（分かるとこから）。詰まったら教科書見ながら一緒に。

### 一番大事なトラップ（必ず通す）

表が埋まったら、こちらから問う：

> 「じゃあ植物って、昼は光合成だけしてる？」

→ 多分「うん」と答える or 一瞬迷う。

ここで**赤マーカー**で表の下に大書き：

> **植物は呼吸を常にしている。昼は呼吸＋光合成、夜は呼吸だけ。**

つづけて図解：太陽の絵を上に描き、昼の植物と夜の植物を 2 体描く。それぞれに気体の出入り矢印を入れる：

- 昼：CO₂吸う（光合成）／O₂出す（光合成）／O₂吸う（呼吸）／CO₂出す（呼吸） → 見かけ上 CO₂吸って O₂出す（光合成 > 呼吸）
- 夜：O₂吸う／CO₂出す（呼吸のみ）

> **見かけの気体出入り**（昼）と**実際にやってること**（両方）の区別が定期テスト頻出ポイント。

### 暗記タグ

- 光合成 = 「**光**って名前ついてる時点で**昼だけ**」
- 呼吸 = 「**人間と同じで死ぬまでずっと**」

## 4. テーマ③：BTB溶液（17:35-17:50）

### 一緒に描く三角関係図

白紙の **右半分** を使う。

1. **色のスケール**を横に描く：

   ```
   黄色 ←――――― 緑 ―――――→ 青
   酸性          中性          アルカリ性
   ```

   - 「酸性=黄、アルカリ性=青、間が緑」を声に出して 3 回唱える
   - 暗記タグ：「**ア**ルカリ性 = **あ**おい」（あ→あ）／酸性は残った黄

2. **CO₂ 量との対応**を下に矢印で：

   ```
   CO₂多い → 水に溶けて酸性寄り → 黄色
   CO₂普通 → 中性 → 緑
   CO₂少ない → アルカリ性寄り → 青
   ```

   - ポイント：「**CO₂は水に溶けると酸っぱい（炭酸）**」 → これがすべての出発点
   - 炭酸水を例に：「コーラ振ると泡（CO₂）出てくるじゃん、あれが溶けてるから酸っぱい」

### オオカナダモ実験図（必修パターン）

定期テスト頻出。試験管 3 本セット：

| 試験管 | 中身 | 光 | 結果（BTB の色） | 解釈 |
|---|---|---|---|---|
| A | 水＋BTB＋息（CO₂入） + オオカナダモ | 当てる | **黄→青** | **光合成で CO₂ 吸った**（呼吸より光合成多い） |
| B | 同上 | 当てない（暗所） | **黄のまま or もっと黄** | **呼吸だけ、CO₂ 増える** |
| C | 同上、オオカナダモなし（対照） | 当てる | **黄のまま** | 光だけでは色変わらない＝植物が原因と証明 |

→ **C（対照実験）の意味**が分かってない子が多い。「C がないと『光だけで変わったかも』と疑える」というロジックを必ず説明。

### 暗記の最終フレーズ

> 「黄＝CO₂多い／青＝CO₂少ない／緑＝ちょうど中間」
> 「光合成は CO₂吸う＝青寄り、呼吸は CO₂出す＝黄寄り」

## 5. 想定問答（詰まりポイント予測）

### Q1: 「光合成と呼吸って同時にやってるの？」
**A（誘導）**: 「いいとこ突くね。同時にやってる。だから昼は表面上 CO₂吸ってるように見えても、実は呼吸で CO₂出してもいる。図に戻ろう」→ 昼の植物の絵に戻る。

### Q2: 「BTBが緑のままって何を意味する？」
**A（誘導）**: 「中性ってことは、CO₂ が増えてもないし減ってもない、てこと。光合成と呼吸が**ちょうど釣り合ってる**状態。じゃあ条件何だと思う？」→ 「ちょっと暗い・薄明るい時」と引き出せたら最高。

### Q3: 「道管と師管、どっち内側だっけ？」
**A（誘導）**: 「迷うやつね。**水**って重いから内側、って覚え方ある。**道**は**水**道の**道**、内側で水運んでる」→ 図の縦切りに戻る。

### Q4（ありそう）: 「葉緑体って葉っぱだけにあるの？」
**A**: 「基本は葉。茎の緑のとこにもちょっとある。根にはない（光当たらないから）」→ 深追いせず次へ。

### Q5（ありそう）: 「ミトコンドリアって何？」
**A**: 「全部の細胞にある、呼吸する場所。葉緑体は植物だけだけど、ミトコンドリアは動物にもある。**僕らも呼吸してるからね**」→ 自分の体と結びつけて記憶を強化。

### 共通リカバリーフレーズ

詰まった時の言葉（暗記しておく）:

- 「これ最初は分かんなくて当たり前」

- 「今わかんなくてOK、図描いたら分かるはず」

## 6. 解き直し用ミニ問題（17:50-17:55 / 5問中 3問選択）

教科書 STEP3 発展は今日入れない。基礎〜標準のみ。

### Q1（道管・師管：基礎）
次の文章の空欄を埋めなさい。
> 植物の茎の内側を通り、根から吸い上げた水や無機養分の通り道を（　①　）といい、外側を通り、葉でつくられた養分の通り道を（　②　）という。両方をまとめて（　③　）という。

**答**: ①道管 ②師管 ③維管束

### Q2（光合成 vs 呼吸：標準）
次のうち、光合成だけが行うものに○、呼吸だけが行うものに△、両方行うものに□をつけなさい。
- (a) 酸素を出す　　　(b) 二酸化炭素を出す　　　(c) 光が必要
- (d) エネルギーをつくる　　(e) 葉緑体で行う　　(f) 24時間行う

**答**: (a)○ (b)△ (c)○ (d)○ (e)○ (f)△

### Q3（BTB：標準）
青色のBTB溶液にストローで息をふきこんだ。色はどう変わるか。理由とともに答えよ。

**答**: 黄色（または緑）に変わる。息に含まれる二酸化炭素が溶けて酸性に近づくため。

### Q4（オオカナダモ実験：頻出）
試験管にBTB溶液（黄色）を入れ、オオカナダモを入れて日光に当てた。しばらくすると BTB溶液は青色に変わった。この理由を述べよ。

**答**: オオカナダモが光合成を行い、溶液中の二酸化炭素を吸収したから（CO₂が減ってアルカリ性に近づいた）。

### Q5（対照実験：思考）
Q4 の実験で、オオカナダモを入れない試験管も同時に用意した理由を答えよ。

**答**: 色の変化がオオカナダモ（の光合成）によるものであることを確かめるため。対照実験。

**運用**: 3 問選ぶ。**Q1必須**（基礎確認）→ 流れ次第で Q2 or Q4 → 余裕あれば Q3 or Q5。
