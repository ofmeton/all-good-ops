## 1. note.com 勝ちパターン

### 7項目サマリ

#### タイトルパターン
- **「数値 + ベネフィット」型が支配的**。「ChatGPTで仕事が10倍速に｜活用法5選」「AI活用108選」「3段階プロンプト」「月3万円」のように、具体的な数字（〇選/〇倍/月〇円）が圧倒的に多い
- **「ツール名 × プラットフォーム × 行動動詞」**もテンプレ化（例：「【生成AI×note】Claudeをマイ編集者に育てよう」）
- **「How-to + 体験談ハイブリッド」が最強**。純How-toより「副業3年目で見えてきた〇〇」「実証報告」のように体験談フレームを被せた方が刺さる
- **避ける**: 「書いてみた」「気になる方はこちら」「やってみた感想」等のフワッとした日記タイトル

#### 構成パターン
- **「共感→価値→行動」3段構成**が王道。冒頭で読者の悩みを代弁（「あぁ、これは生成AIっぽいなぁ…」のような感覚から入る）し、自分の失敗談 → 解決策 → テンプレ/手順という流れ
- **108選/5選などの「番号付き網羅型」は CV と SNS シェアの両方を稼ぐ**。スマートバンクの108選はほぼ全文無料公開で採用導線として機能
- **noteの「推奨」は 1次情報・生の体験・失敗談**（深津氏明言）。AI生成だけの抽象論は推薦から外れる

#### 価格帯
- **500円**: お試し導入・心理障壁低（「カフェラテ1杯」訴求）。1本目はここが定番
- **800〜980円**: 主戦場。月3万円狙いなら 980円 × 31本/月が標準モデル
- **1,280〜1,980円**: テンプレ・チェックリスト・プロンプト集を「再現可能な形」で同梱した時のみ正当化される
- **2,000円超**: 「ステップ解説 + 実テンプレ + 続報リンク」のセット販売型

#### 有料化の境目
- **無料 = 「なるほど」、有料 = 「これで動ける」**が境目の合言葉
- 無料部分で **(a) 悩みの代弁、(b) 得られる未来の提示、(c) 信頼性の証明（経歴・実数値）** を必ず置く
- 関美穂子氏のClaude記事は「手法の全体像と設定ステップ5つまでを無料、評価チェックリスト39項目を800円」にする「テンプレ・チェックリストは有料」分岐がきれい
- 全文無料 + 採用/コミュニティ/LINE誘導モデル（スマートバンク108選・KEITO氏Procevo）も有効

#### ターゲット比率
- **非エンジニア向けの方が単価・購買頻度ともに上**。エンジニア向けはGitHub等に流出するため、note 有料記事のメインは **中小事業者・士業・副業勢・広報職・編集職** が刺さる層
- 売れている非エンジニア向け記事は「コードを書かずに〇〇」「事務作業のラストワンマイル」「中小企業のDX」など、業務用語でターゲット明示型

#### 画像活用
- **スクショ多用 + 番号付き手順 + タグクラウド**がデフォルト。KEITO氏のClaude記事はセクションごとに操作画面3〜4枚
- **テキストオンリーは伸びない**。AI活用108選は箇条書き・テーブル・スライド資料形式で視認性を確保
- ヘッダー画像 + アイキャッチは購入率に直結（noteのSNSシェア表示にも影響）

#### 避けるべきパターン
- **AIっぽさ全開の総論記事**（事例ゼロ、抽象論のみ、です・ます直訳調）
- 一次情報なし・体験談なしの「ChatGPTでできる10のこと」型 → noteアルゴリズム的にも推奨対象外
- 曖昧なCTA（「気になる方はこちら」「ご興味あれば」）
- 純エッセイ型（感情吐露だけ）と純知識型（読者主語なし）

### 応用すべき勝ちパターン Top 5
1. **「数値 + 非エンジニア向け業務用語」タイトル固定**（例：「中小事業者がClaudeで請求書作成を15分→3分にした手順」）
2. **無料 = 全体像と気づき / 有料 = チェックリスト・プロンプト・テンプレ集**
3. **構成は「失敗談 → 試した3手 → 残った1手 → テンプレ配布」**
4. **スクショ + 番号付き手順を全記事必須化**（1記事あたり画面キャプチャ最低6〜8枚）
5. **ofmeton 名義は「業務効率化の現場記録ストック型」、はぐりんと役割明確化**

## 2. X（Twitter）勝ちパターン

### 7項目サマリ

#### フック1行目
- **【】記号 + 強ワード（速報/朗報/驚愕/超時短/必見）**が圧倒的定番
- **数値先出し**（「3ヶ月放置で2倍」「月750万円」「30分で構築」）と**結論先出し**（「これ全員やるべき。」断定文）が並走で強い
- **問いかけ型は単体だと弱い**。「結末提示」型の方が伸びる
- 「最初の15文字で勝負が決まる」が日本AI系インフルエンサーの共通認識

#### 投稿形式
- **単発投稿 + 画像/動画** が伸びの主流。スレッドはサブ手段
- バズ層は「**1枚目で完結 → 2枚目以降は深掘り**」のキャリーオーバー設計
- スレッドは長文ノウハウ（プロンプト全公開系）でnote誘導前提に作るのがセオリー。「3ツイート構成（フック→ボディ→CTA）」が標準テンプレ

#### 画像活用
- **比較表/Before-After画像**が拡散源として最強格
- **手書き風・薄い色面の図解（チャエン式）**が「AIっぽくない・読みやすい」で好まれる
- **スクショ + 赤枠/矢印アノテーション**（Cursor/Claude Code 操作画面）はエンジニア層に強い
- 「1280x720級の動画」が滞在時間でアルゴリズムに刺さる

#### テーマ別反応
- 強い順: **(a) プロンプト集全公開 > (b) 業務自動化のBefore-After > (c) Cursor/Claude Code 実装デモ > (d) AIで稼ぐ系**
- **「コピペで使える」「全公開」「完全保存版」**がブースト語
- 「AIで稼ぐ」単独テーマはエンジニア層にスベりやすい。「業務自動化 + 副業」のように実用 × 経済合理性で二段重ねが伸びやすい

#### エンジニア発信者 vs 非エンジニア発信者
- **エンジニア系**: Claude Code/Cursor 実装デモ、自作OSS、技術的にスマートな小ネタ。拡散層は技術クラスタ中心、いいね多めだがRTは少なめ
- **非エンジニア系（チャエン/枌谷力/深津貴之）**: 「**非エンジニアでも分かる**」という"翻訳役"ポジション。RT・引用ともに爆発しやすい
- ofmeton ターゲットに刺さるのは後者ポジション。「エンジニアだけど非エンジニア向けに翻訳する人」というハイブリッドが狙い目

#### note送客
- **「無料部分を分解 → X単発投稿の連投ネタにする」フロー**が定石
- チャエン氏のnote記事「AI駆動のX運用完全攻略本」5.6万字 は X複数投稿で誘導継続
- **CTA前にいいね/RT/フォローを直接ねだらない**。「↓詳細とプロンプト」だけで自然導線
- 深津式は **noteで体系化 → Xで断片配布** の往復構造で長期ストック化

#### 避けるべきパターン
- **AI生成感の強い文体**（絵文字過剰、整いすぎた見出し、「いかがでしたでしょうか」調）
- **流行乗っかりの実演投稿（Grok等）でクリエイター層を刺激**して炎上した事例あり
- **「神プロンプト」「これだけで人生変わる」誇大煽り**は短期は伸びるが信頼コスト高
- **AI使用を隠蔽**したクリエイティブ投稿は2025年以降特に反感を買う

### 応用すべきX運用 Top 5戦術
1. **「エンジニアだけど非エンジニア語で翻訳する人」ポジション確立**
2. **Before-After画像1枚 + 数値見出しの単発投稿を主軸**
3. **「コピペで使える」プロンプト + 全公開**（深掘りは note 有料化）
4. **ネタはターゲット業務に張り付ける**（「行政書士×Claude」「中小工務店×AI」等）
5. **AI使用の透明性 + 制作プロセス公開**

## 3. Instagram 勝ちパターン

### 7項目サマリ

#### 投稿形式の分布
AI/業務効率化ジャンルは **カルーセル（8-10枚）が主戦力**。8-10枚がエンゲージメント最大化のスイートスポット。リールは「AI活用例の画面録画＋テロップ」でフォロワー獲得の入口、カルーセルが保存・プロフ流入の核という役割分担。単一画像は告知用途以外ほぼ使われていない。保存率15%超でリーチが伸びる構造。

#### 1枚目（カバー）のデザイン
- **巨大な数字 or 強ワード**（「3つの裏ワザ」「これ知らない人ヤバい」「秒で終わる」）
- **背景は単色 or ベタ塗り**（黒/ネイビー/朱赤/黄＋黒文字が定番）
- **人物写真は使うアカウント / 完全タイポのみアカウント に二極化**
- **左下に「保存推奨」マークや矢印で続きを示唆**
- フォント: 太いゴシック（Noto Sans / 源ノ角ゴシック Heavy）+ 部分的に蛍光カラー or 黄色マーカー

#### カルーセル本編
王道は **9枚構成の役割分担**:
- 1枚目: フック
- 2-7枚目: 展開（分解 → 比較 → 実例 → ミニ課題）
- 8枚目: まとめ（3-5行で再圧縮）
- 9枚目: CTA（保存/フォロー/プロフィール遷移）

主流は **「左に見出し＋右にスクショ」or「上に問い＋下に答え」の2分割**。スクショは枠線・ドロップシャドウで浮かせる。Before/After比較・ビフォーアフター表・チェックリスト形式が保存率高い。

#### テーマ別人気
1. ChatGPTプロンプト集 N選（保存特化）
2. 業務効率化 tips（Excel/メール/議事録/資料作成）
3. AI副業 / 月X万円系
4. Notion×AI / 仕事整理
5. 新モデル/新機能の即解説
6. AI画像生成プロンプト

ofmeton 中小事業者・士業・コンサルターゲットは **2・4** が最適。

#### AI画像生成系との差別化
- **画面スクショ多用**で「実際に使えそう感」
- **数字・固有名詞・ツール名**を出してSEO的にも検索される
- **顔出し or イニシャル・アバター**でAI感を消す
- **動画はキーボード操作 or 画面録画**

ofmeton は「実装の裏側まで見せる」ポジションが取れる（多くのtips系は表面のプロンプト紹介止まり）。

#### 外部送客動線
- **プロフィール**: 1行目「○○を発信」、2行目「実績/権威性」、3行目「note/メルマガ」、URLは Linktree 不使用で **note直リンク** が多い
- **ハイライト**: 「自己紹介」「実績」「サービス」「note記事」「お問い合わせ」の5カテゴリ標準
- **投稿内CTA**: 9枚目で「プロフィールから○○」と明示。キャプション直接URLは無効
- **DM自動応答**で資料配布（ManyChat等）→ ノウハウPDFを DMで配って note 有料記事へ送る二段構成が増加中

#### 避けるべきパターン
- **文字過多**（1スライド20字以上で離脱率上昇）
- **Canvaデフォルトテンプレ丸出し**
- **AI生成のサムネイル写真**（指が6本、目が虚ろ）
- **派手なグラデーション**（紫×ピンク×水色のいかにも"AI"な配色）
- **絵文字過多**（特に🚀✨🔥の3点セット）
- **無意味な装飾枠**（吹き出し・スター・キラキラ）

### 応用すべきInstagram戦術 Top 5
1. **カルーセル8-10枚を主軸にする**（1枚目テンプレ固定 / 2-7実装スクショ / 8まとめ / 9 note送客CTA）
2. **「実装の裏側まで見せる」差別化ポジション**
3. **デザインシステムを固定化**（Noto Sans Heavy / 黒・濃紺・朱赤 / アクセント黄色 / Figma 自作テンプレ）
4. **ハイライト5カテゴリ固定**（自己紹介 / AI制作事例 / 業務効率化tips / note記事 / お仕事依頼）
5. **投稿テーマの3軸ローテ**（① プロンプト集N選 / ② AI実装事例 / ③ ツール最新動向）

## 4. 統合した「ofmeton 発信のコア戦略」

### 一行サマリ
> **エンジニアだけど、非エンジニアの言葉で「業務 × 短縮時間」を翻訳する実装者**

### 3媒体の役割
- **X**: Before-After 単発投稿で「翻訳役」演出 → note 送客
- **Instagram**: カルーセル9枚で「実装の裏側」ビジュアル特化 → note + プロフ送客
- **note**: 「失敗談→試行→残った1手→テンプレ配布」構成、500-980円帯、月1有料+月3-5無料

### 共通の禁止事項（content-reviewer 必須チェック）
1. AI 感の強い文体（「いかがでしょうか」「ぜひお試しください」等）
2. テキストオンリー（画像なし）
3. 専門用語の非注釈濫用
4. ターゲット業務が不明確（誰の何が楽になるかが冒頭にない）
5. 「神プロンプト」「これだけで人生変わる」誇大煽り
6. AI使用の隠蔽

### 共通のテンプレ言語
- 「行政書士の見積書作成、Claude で 3時間 → 5分」
- 「中小工務店の提案資料、AI で 半日 → 30分」
- 「士業の顧客対応定型文、コピペで 10件 → 1セット」
