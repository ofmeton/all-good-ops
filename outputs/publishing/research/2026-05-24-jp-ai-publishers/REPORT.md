# Executive Summary

## 4. ofmeton 発信戦略への落とし込み（提案）

| 柱 | Tier 1/2/3 マッピング |

### 3媒体への配分案

- **X**: T1-4（失敗談）の単発投稿 + T1-3（ROI数字 Before-After）→ 拡散
- **Instagram**: T2-4 のカルーセル独占枠（9枚で1業種ガイド）→ ブランド構築
- **note**: T2-3（1万字級）で T1-1 を業種別フルガイド化 → 月500-980円で有料

### 3ヶ月で押さえる優先順位

1. **月1**: T1-4（失敗談先行型）を X 軸で量産 → ofmeton の声をつける
2. **月2**: T1-1（業種別SOP）を 3 業種で note 有料化 → 収益化開始
3. **月3**: T2-4（Instagram カルーセル）を立ち上げ → ブランド構築

## 5. 補足: データの限界

- twitterapi.io advanced_search は **1ハンドル最大100件/直近90日**で取得。投稿頻度が高い人ほどリプライ・RT が多くなりサンプル偏りあり
- 「書かれていない」は **X 上での観測**のみ。note・YouTube・Voicy 等の媒体は本調査範囲外
- @kosuke_agos / @ai_jitan のように選定軸（Claude/AI業務自動化）からズレるアカウントが混ざっており、実質は **8 名の発信者** + 2名のバズ参考が母集団
- 投稿テキストのみ精読。引用画像・添付動画の内容は未解析
