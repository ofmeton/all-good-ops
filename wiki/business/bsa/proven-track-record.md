## 1. 実績として提案文に書ける項目

### A. 広告運用（Lifunext 時代 / 2022.08-2025.03）

**A-1. 寝具D2C（枕・布団）CPA改善**
- 業種: 寝具D2C（枕1.5万円・布団5万円のサブスク／単発販売）
- 媒体: Google リスティング広告
- 期間: 2023年04月〜07月（3ヶ月）
- 体制: 主担当（単独運用）
- 成果: **CPA 約9.4万円 → 1.5万円（84%削減）**
- 一行表現例: 「寝具D2C（枕・布団）の Google リスティング運用で CPA 84%削減（約9.4万円→1.5万円、3ヶ月、主担当）」
- 注意: 「LP改修と同時設計」「ヘルスケアLP」とは書かない（事実と異なる）

**A-2. 並行運用キャパシティ**
- 期間: 2022.08〜2024.01
- 担当数: 最大6社並行（主担当として単独運用）
- 担当業種: NPO（寄付） / SaaS（BtoB EC広告管理ツール） / 美容（D2Cシャンプーサブスク） / 寝具（枕・布団） / マットレス（NELL）
- 一行表現例: 「最大6社を並行で広告運用（NPO / SaaS / 美容 / 寝具）。社内顧客満足度1位」

**A-3. NELL 単月3000万円PoC運用**
- 業種: マットレス D2C（NELL）
- 媒体: 6媒体横断（Yahoo / Google LIS DIS / Meta / X / LINE / TikTok）
- 期間: 2023年09月（単月）

### B. データ自動化・基盤構築（Lifunext オペレーション部時代 / 2024.02-2025.03）

**B-1. BigQuery 導入**
- 業種: 広告代理店内部データ基盤
- 体制: 3名チーム主導（自身が起案・設計・実装リード）
- 期間: 3ヶ月
- 成果: スプシ毎日クラッシュ → 0回。Looker / LLM 接続拡張を視野
- 一行表現例: 「広告代理店の社内データ基盤を BigQuery で再構築（3名チーム主導・3ヶ月）」

**B-2. T&Dカスタマイザ自動化**
- 内容: Excel 多段条件分岐関数で全地域×空港別×価格別の組み合わせを自動生成
- 成果: **5-6人×10時間 → 1人×1時間（98%削減）**
- 一行表現例: 「広告クリエイティブ生成業務を Excel 多段関数で自動化、工数 98%削減」

**B-3. Wチェック自動化（GAS）**
- 内容: 入稿指示データと実入稿データの照合を Google Apps Script で自動化
- 成果: **工数 90%削減**
- 一行表現例: 「広告入稿のWチェックを GAS で自動化、工数 90%削減」

## 3. 標準スタック対応可否（提案不可案件の判定基準）

工藤陸が **BSA 標準スタック（Next.js + microCMS + Cloudflare Pages or Vercel Pro）** で実装する案件を主軸とする。
以下の特殊要件案件は **提案前に「実装可能性」を要確認**。

| 要件 | 対応可否 | 備考 |
|---|---|---|
| Next.js + Tailwind + microCMS | ◎ 標準 | BSA メイン |
| 静的HTML+CSS のみ | ◎ | L1 / L4 |
| WordPress 通常テーマカスタム | △ | 1社目 SI 経験ありだが本人実装は限定的。要確認 |
| **WordPress + SWELL 特化案件** | × **未経験** | 提案前に「経験ありますか？」を本人に確認必須 |
| **WordPress + 賢威** | × 未経験 | 同上 |
| Shopify | △ 副業で運営経験あり、商品管理レベル | テーマ開発は要確認 |
| **18禁ブログ・アダルト案件** | × 受注しない | ofmeton portfolio に実績露出不可・BSA 継続運用に活かせない |
| WebGL / Three.js / 凝った 3D 演出 | × | 標準 LP/HP の演出引き出しを超える要件は受けない |
| 多言語サイト | △ | next-intl で実装可能だが工数追加 |
